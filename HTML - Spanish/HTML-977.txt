			<h1 id="firstHeading" class="firstHeading" lang="es">Clima mediterráneo</h1>

</div><div class="thumb tright"><div class="thumbinner" style="width:202px;"><a href="/wiki/Archivo:Distribucion_mediterraneo.png" class="image"><img alt="" src="https://upload.wikimedia.org/wikipedia/commons/thumb/5/5c/Distribucion_mediterraneo.png/200px-Distribucion_mediterraneo.png" width="200" height="141" class="thumbimage" srcset="https://upload.wikimedia.org/wikipedia/commons/thumb/5/5c/Distribucion_mediterraneo.png/300px-Distribucion_mediterraneo.png 1.5x, https://upload.wikimedia.org/wikipedia/commons/thumb/5/5c/Distribucion_mediterraneo.png/400px-Distribucion_mediterraneo.png 2x" data-file-width="3085" data-file-height="2181" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Archivo:Distribucion_mediterraneo.png" class="internal" title="Aumentar"></a></div>Distribución del clima mediterráneo en el mundo.</div></div></div>
<p>El <b>clima mediterráneo</b> se caracteriza por  <a href="/wiki/Invierno" title="Invierno">inviernos</a> templados y lluviosos y <a href="/wiki/Verano" title="Verano">veranos</a> secos y calurosos, con otoños y primaveras variables, tanto en temperaturas como en precipitaciones. El nombre lo recibe del <a href="/wiki/Mar_Mediterr%C3%A1neo" title="Mar Mediterráneo">mar Mediterráneo</a>, área donde es típico este clima y adquiere mayor extensión geográfica, pero también está presente en otras zonas del planeta, aunque con variaciones en cuanto a la distribución de las temperaturas. tienen este tipo de clima en <a href="/wiki/California" title="California">California</a>, <a href="/wiki/Baja_California" title="Baja California">Baja California</a>, sur de <a href="/wiki/Sud%C3%A1frica" title="Sudáfrica">Sudáfrica</a>, centro de <a href="/wiki/Chile" title="Chile">Chile</a> y este de <a href="/wiki/Australia" title="Australia">Australia</a>.
</p><p>Las <a href="/wiki/Lluvia" title="Lluvia">lluvias</a> no suelen ser muy abundantes. Pero la característica principal es que éstas no se producen en verano. Las temperaturas se mantienen, en promedio, todos los meses por encima de los 0ºC.
</p>
<h2><span class="mw-headline" id="Vegetaci.C3.B3n">Vegetación</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Clima_mediterr%C3%A1neo&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Editar sección: Vegetación">editar</a><span class="mw-editsection-divider"> · </span><a href="/w/index.php?title=Clima_mediterr%C3%A1neo&amp;action=edit&amp;section=1" title="Editar sección: Vegetación">editar código</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tright"><div class="thumbinner" style="width:202px;"><a href="/wiki/Archivo:Bosque_escler%C3%B3filo,_Chile_001_%E2%80%94_Inao_V%C3%A1squez.jpg" class="image"><img alt="" src="https://upload.wikimedia.org/wikipedia/commons/thumb/0/05/Bosque_escler%C3%B3filo%2C_Chile_001_%E2%80%94_Inao_V%C3%A1squez.jpg/200px-Bosque_escler%C3%B3filo%2C_Chile_001_%E2%80%94_Inao_V%C3%A1squez.jpg" width="200" height="150" class="thumbimage" srcset="https://upload.wikimedia.org/wikipedia/commons/thumb/0/05/Bosque_escler%C3%B3filo%2C_Chile_001_%E2%80%94_Inao_V%C3%A1squez.jpg/300px-Bosque_escler%C3%B3filo%2C_Chile_001_%E2%80%94_Inao_V%C3%A1squez.jpg 1.5x, https://upload.wikimedia.org/wikipedia/commons/thumb/0/05/Bosque_escler%C3%B3filo%2C_Chile_001_%E2%80%94_Inao_V%C3%A1squez.jpg/400px-Bosque_escler%C3%B3filo%2C_Chile_001_%E2%80%94_Inao_V%C3%A1squez.jpg 2x" data-file-width="1024" data-file-height="768" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Archivo:Bosque_escler%C3%B3filo,_Chile_001_%E2%80%94_Inao_V%C3%A1squez.jpg" class="internal" title="Aumentar"></a></div>Paisaje de clima mediterráneo en el área metropolitana de <a href="/wiki/Santiago" class="mw-redirect" title="Santiago">Santiago</a> (<a href="/wiki/Chile" title="Chile">Chile</a>)</div></div></div>
<p>La <a href="/wiki/Vegetal" title="Vegetal">vegetación</a> está formada por <a href="/wiki/%C3%81rbol" title="Árbol">árboles</a> de hoja perenne y matorrales.
</p><p>En las zonas con este clima es donde se ha desarrollado tradicionalmente la llamada <i>trilogía mediterránea</i>: <a href="/wiki/Trigo" title="Trigo">trigo</a>, <a href="/wiki/Vid" title="Vid">vid</a> y <a href="/wiki/Olivo" title="Olivo">olivo</a>. Este último es un árbol que únicamente se cultiva en zonas que tienen este clima. Las zonas de clima mediterráneo tienen muy desarrollada la <a href="/w/index.php?title=Agricultura_de_regad%C3%ADo&amp;action=edit&amp;redlink=1" class="new" title="Agricultura de regadío (la página no existe)">agricultura de regadío</a> produciéndose gran cantidad de <a href="/wiki/Fruta" class="mw-redirect" title="Fruta">frutas</a> (<a href="/wiki/Uva" title="Uva">uvas</a>, <a href="/wiki/Lim%C3%B3n" class="mw-redirect" title="Limón">limones</a>, <a href="/w/index.php?title=Albaricoque&amp;action=edit&amp;redlink=1" class="new" title="Albaricoque (la página no existe)">albaricoques/damascos</a>, <a href="/w/index.php?title=Melocot%C3%B3n&amp;action=edit&amp;redlink=1" class="new" title="Melocotón (la página no existe)">melocotones</a>, <a href="/w/index.php?title=Cereza&amp;action=edit&amp;redlink=1" class="new" title="Cereza (la página no existe)">cerezas</a>, <a href="/w/index.php?title=Ciruela&amp;action=edit&amp;redlink=1" class="new" title="Ciruela (la página no existe)">ciruelas</a>, <a href="/w/index.php?title=N%C3%ADspero&amp;action=edit&amp;redlink=1" class="new" title="Níspero (la página no existe)">nísperos</a>, etc.) y <a href="/wiki/Hortaliza" title="Hortaliza">hortalizas</a> (<a href="/wiki/Tomate" title="Tomate">tomates</a>, <a href="/wiki/Solanum_tuberosum" title="Solanum tuberosum">papas/patatas</a>, <a href="/wiki/Berenjena" title="Berenjena">berenjenas</a>, <a href="/wiki/Cucurbita_pepo" title="Cucurbita pepo">zapallo/calabacín</a>, <a href="/wiki/Cebolla" title="Cebolla">cebollas</a>, <a href="/wiki/Ajo" title="Ajo">ajos</a>, <a href="/wiki/Zanahoria" title="Zanahoria">zanahorias</a>, etc.), quedando en el <a href="/wiki/Agricultura_de_secano" title="Agricultura de secano">secano</a> el ya mencionado olivo junto a otras especies como <a href="/w/index.php?title=Almendro&amp;action=edit&amp;redlink=1" class="new" title="Almendro (la página no existe)">almendros</a> y <a href="/w/index.php?title=Algarrobos&amp;action=edit&amp;redlink=1" class="new" title="Algarrobos (la página no existe)">algarrobos</a>.
</p>

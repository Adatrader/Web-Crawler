<p><a href="/wiki/Wikipedia" title="Wikipedia">Wikipedia</a> es un <a href="/wiki/Wiki" title="Wiki">wiki</a>. Esto significa que cualquier persona puede editar fácilmente cualquier página (excepto las <a href="/wiki/Wikipedia:Pol%C3%ADtica_de_protecci%C3%B3n" title="Wikipedia:Política de protección">protegidas</a>), y sus cambios serán visibles inmediatamente. Aquí aprenderás <b>cómo se edita una página</b>. Las indicaciones que se dan sirven para cualquier tipo de página, aunque principalmente se editan artículos.
</p><p>Se explican algunas cosas sobre la sintaxis: cómo <a href="#Enlaces">crear enlaces</a>, los principales <a href="#Formatos_de_texto">formatos de texto</a>, estructurar el contenido en <a href="#Secciones">secciones</a>, cómo <a href="#Archivos">insertar imágenes</a>, hacer <a href="#Listas">listas</a> y <a href="#Tablas">tablas</a>, agregar <a href="#Referencias">referencias</a>, <a href="#Plantillas">plantillas</a>, <a href="#Categorías">categorías</a> e <a href="#Interwikis">interwikis</a>, y algunas cosas más avanzadas como las <a href="#Fórmulas">fórmulas</a> o <a href="#Líneas_de_tiempo">líneas de tiempo</a>.
</p><p>Además se detallan algunas peculiaridades de edición para <b><a href="#Otros_tipos_de_páginas">otros tipos de páginas</a></b> distintas a los artículos. Por último hay una serie de <b><a href="#Consejos_y_recomendaciones">consejos y recomendaciones</a></b> a seguir.
</p><p>Recuerda realizar las pruebas en la <a href="/wiki/Wikipedia:Zona_de_pruebas/10" title="Wikipedia:Zona de pruebas/10">zona&#160;de&#160;pruebas</a>. Después, puedes buscar una de las muchas <a href="/wiki/Ayuda:C%C3%B3mo_puedes_colaborar" title="Ayuda:Cómo puedes colaborar">tareas por hacer</a> y <a href="/wiki/Wikipedia:S%C3%A9_valiente_editando_p%C3%A1ginas" class="mw-redirect" title="Wikipedia:Sé valiente editando páginas">ser valiente editando páginas</a>. ¡Al hacer tu primera edición te conviertes en <a href="/wiki/Wikipedia:Wikipedista" class="mw-redirect" title="Wikipedia:Wikipedista">wikipedista</a>!
</p>

<h2><span class="mw-headline" id="Primeros_pasos">Primeros pasos</span></h2>
<p>Para editar Wikipedia no es necesario estar registrado, aunque sí es cierto que registrarse conlleva una <a href="/wiki/Ayuda:Registro" title="Ayuda:Registro">serie de ventajas</a>. Decidas registrarte o no, el proceso de editar un artículo puede resumirse en <b>tres pasos</b>.
</p>
<div class="thumb tmulti tnone center"><div class="thumbinner" style="width:762px;max-width:762px"><div class="tsingle" style="float:left;margin:1px;width:252px;max-width:252px"><div class="thumbimage"><a href="/wiki/Archivo:Tutorial_de_edici%C3%B3n,_paso_1.png" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/ae/Tutorial_de_edici%C3%B3n%2C_paso_1.png/250px-Tutorial_de_edici%C3%B3n%2C_paso_1.png" decoding="async" width="250" height="137" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/ae/Tutorial_de_edici%C3%B3n%2C_paso_1.png/375px-Tutorial_de_edici%C3%B3n%2C_paso_1.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/ae/Tutorial_de_edici%C3%B3n%2C_paso_1.png/500px-Tutorial_de_edici%C3%B3n%2C_paso_1.png 2x" data-file-width="838" data-file-height="460" /></a></div><div class="thumbcaption" style="clear:left"><b>Paso 1</b>. <h3><span id="C.C3.B3mo_empezar_una_p.C3.A1gina"></span><span class="mw-headline" id="Cómo_empezar_una_página">Cómo empezar una página</span></h3>
<div class="noprint AP rellink"><span style="font-size:88%">Artículos principales:</span>&#32;<i><a href="/wiki/Ayuda:C%C3%B3mo_empezar_una_p%C3%A1gina" title="Ayuda:Cómo empezar una página"> Ayuda:Cómo empezar una página</a></i><span style="font-size:88%">&#32;y&#32;</span><i><a href="/wiki/Wikipedia:Convenciones_de_t%C3%ADtulos" title="Wikipedia:Convenciones de títulos"> Wikipedia:Convenciones de títulos</a></i>.</div>
<p>Existe una página con una explicación detallada sobre <b><a href="/wiki/Ayuda:C%C3%B3mo_empezar_una_p%C3%A1gina" title="Ayuda:Cómo empezar una página">cómo empezar una página</a></b>. Básicamente hay dos formas:
</p>
<ol><li>Mientras editas el texto de una página existente puedes poner algo entre dobles corchetes como, por ejemplo, <code>[[mi página]]</code>. Cuando guardes la página, aparecerá un enlace azul si la página ya existe, o rojo si aún no. En el segundo caso, pulsa sobre él e irás a una nueva página que puedes editar. Reemplaza el texto «Escribe aquí el contenido de la página» con el texto que desees.</li>
<li>Teclea el título de tu nueva página precedido de la cadena «http://es.wikipedia.org/wiki/» en el campo «URL» de tu navegador y pulsa la <a href="/wiki/Tecla_Enter" class="mw-redirect" title="Tecla Enter">tecla Enter</a>. Si todo resulta bien, llegarás a una página con el título que usaste y podrás empezar a editarla.</li></ol>
<p>El modo recomendado es el primero. Se recomienda esto para que las páginas no queden sueltas, sino que haya algún enlace entre ellas y sea más fácil que el usuario las encuentre. A la hora de nombrar las páginas se recomienda seguir las <b><a href="/wiki/Wikipedia:Convenciones_de_t%C3%ADtulos" title="Wikipedia:Convenciones de títulos">convenciones de títulos</a></b>.
</p>
<h2><span class="mw-headline" id="Enlaces">Enlaces</span></h2>
<p>En <a href="/wiki/Internet" title="Internet">Internet</a> se navega entre páginas a través de enlaces. En Wikipedia se sigue el mismo sistema. Existen principalmente dos tipos de enlaces: internos y externos.
</p>
<h3><span class="mw-headline" id="Enlaces_internos">Enlaces internos</span></h3>
<div class="thumb tright"><div class="thumbinner" style="width:302px;"><a href="/wiki/Archivo:Tutorial_de_edici%C3%B3n,_enlaces_internos.png" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/0/00/Tutorial_de_edici%C3%B3n%2C_enlaces_internos.png/300px-Tutorial_de_edici%C3%B3n%2C_enlaces_internos.png" decoding="async" width="300" height="179" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/0/00/Tutorial_de_edici%C3%B3n%2C_enlaces_internos.png 1.5x" data-file-width="400" data-file-height="239" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Archivo:Tutorial_de_edici%C3%B3n,_enlaces_internos.png" class="internal" title="Aumentar"></a></div>Si no quieres introducir los dobles corchetes (<tt>[[&#160;]]</tt>) a mano, puedes crear los enlaces utilizando el asistente. Selecciona la palabra a enlazar con el ratón y presiona el botón <img alt="Enlace" src="//upload.wikimedia.org/wikipedia/commons/4/48/Vector_toolbar_insert_link_button.png" decoding="async" width="28" height="28" data-file-width="28" data-file-height="28" /> de la <a href="/wiki/Ayuda:Barra_de_herramientas" class="mw-redirect" title="Ayuda:Barra de herramientas">barra de herramientas</a>. Esto abrirá un formulario donde indicarás el título de artículo destino y el título del enlace.</div></div></div>
<div class="noprint AP rellink"><span style="font-size:88%">Artículo principal:</span>&#32;<i><a href="/wiki/Ayuda:Tutorial_(enlaces_internos)" title="Ayuda:Tutorial (enlaces internos)"> Ayuda:Tutorial (enlaces internos)</a></i></div>
<p>Los <b>enlaces internos</b> son aquellos que llevan a otras páginas de Wikipedia. Los enlaces internos se crean siempre que el artículo destino aporte información de valor al artículo actual, permitiendo al lector navegar entre ambos.
</p><p>Por ejemplo, si estamos escribiendo sobre los planetas del <a href="/wiki/Sistema_solar" title="Sistema solar">sistema solar</a>:
</p>
<center><code>[...] los planetas del sistema solar son: Mercurio, Venus, Tierra...</code></center>
<p>Quizás nos interese activar los enlaces internos hacia los artículos de cada planeta. En ese caso, pondremos dobles corchetes alrededor de cada palabra que queramos convertir en enlace:
</p>
<center><code>[...] los planetas del sistema solar son: [[Mercurio]], [[Venus]], Tierra...</code></center>
<p>Automáticamente, las palabras actuarán como enlaces internos. Si el artículo existe, las palabras saldrán en azul; si el artículo no existe, las palabras saldrán en rojo. En este caso existen los dos artículos enlazados:
</p>
<center><code>[...] los planetas del sistema solar son: <a href="/wiki/Mercurio" class="mw-disambig" title="Mercurio">Mercurio</a>, <a href="/wiki/Venus" class="mw-disambig" title="Venus">Venus</a>, Tierra...</code></center>
<p>Pero hay un problema. Mercurio es un planeta, pero también un elemento químico. Venus es un planeta, pero también una diosa de la mitología, etc. Por ello, <a href="/wiki/Mercurio" class="mw-disambig" title="Mercurio">Mercurio</a> y <a href="/wiki/Venus" class="mw-disambig" title="Venus">Venus</a> son <a href="/wiki/Wikipedia:Desambiguaci%C3%B3n" class="mw-redirect" title="Wikipedia:Desambiguación">páginas de desambiguación</a> y contienen una lista de enlaces a todos los posibles significados de esos términos. Bastará consultar esas páginas para saber el título exacto de los artículos sobre el planeta Mercurio y Venus, en este caso: <a href="/wiki/Mercurio_(planeta)" title="Mercurio (planeta)">Mercurio (planeta)</a> y <a href="/wiki/Venus_(planeta)" title="Venus (planeta)">Venus (planeta)</a>. Entonces crearemos los enlaces de este modo (con una barra vertical «|» entre el título del artículo y el título del enlace):
</p>
<center><code>[...] los planetas del sistema solar son: [[Mercurio (planeta)|Mercurio]], [[Venus (planeta)|Venus]], Tierra...</code></center>
<p>Y tendremos, ahora sí, los enlaces apuntando a los artículos correctos.
</p>
<center><code>[...] los planetas del sistema solar son: <a href="/wiki/Mercurio_(planeta)" title="Mercurio (planeta)">Mercurio</a>, <a href="/wiki/Venus_(planeta)" title="Venus (planeta)">Venus</a>, Tierra...</code></center>
<p>Generalmente, los artículos que comparten un mismo título no son frecuentes, por lo que bastará con introducir los corchetes y listo (es el caso de <a href="/wiki/Sistema_solar" title="Sistema solar">sistema solar</a> y <a href="/wiki/Tierra" title="Tierra">Tierra</a>).
</p>
<center><code>[...] los planetas del [[sistema solar]] son: [[Mercurio (planeta)|Mercurio]], [[Venus (planeta)|Venus]], [[Tierra]]...</code></center>
<p>Y obtendrás:
</p>
<center><code>[...] los planetas del <a href="/wiki/Sistema_solar" title="Sistema solar">sistema solar</a> son: <a href="/wiki/Mercurio_(planeta)" title="Mercurio (planeta)">Mercurio</a>, <a href="/wiki/Venus_(planeta)" title="Venus (planeta)">Venus</a>, <a href="/wiki/Tierra" title="Tierra">Tierra</a>...</code></center>
<p>Agrega las conexiones internas que creas necesarias, pero evita abusar de los enlaces: usa solamente los que sean realmente necesarios y no enlaces la misma palabra más de una vez en una misma sección del artículo.
</p>
<h4><span id="Enlaces_a_subp.C3.A1ginas"></span><span class="mw-headline" id="Enlaces_a_subpáginas">Enlaces a subpáginas</span></h4>
<p>Los enlaces internos que comiencen por una <a href="/wiki/Barra_oblicua" class="mw-redirect" title="Barra oblicua">barra oblicua</a> (/) son interpretados como dirigidos a una subpágina. Así <tt>[[/Ejemplo]]</tt> dará el mismo resultado que <tt>[[Ayuda:Cómo se edita una página/Ejemplo|/Ejemplo]]</tt>. Si lo que deseas es que se interprete literalmente debes añadir al comienzo del mismo <a href="/wiki/Dos_puntos" title="Dos puntos">dos puntos</a>, así: <tt>[[:/Ejemplo]]</tt>. Los dos puntos no son visibles en el enlace, lo que permite su integración natural en el texto. Otra alternativa es usar la <a href="/w/index.php?title=Entidad_HTML&amp;action=edit&amp;redlink=1" class="new" title="Entidad HTML (aún no redactado)">entidad HTML</a> correspondiente al carácter de la barra oblicua (<tt>&amp;#47;</tt>). Por ejemplo, <tt>[[&amp;#47;Ejemplo]]</tt> será interpretado literalmente.
</p><p>También puede crear un enlace local a través de una página de <span class="plainlinks"><a class="external text" href="https://www.mediawiki.org/wiki/Help:Extension:GlobalUserPage">usuario global</a></span> (Meta) usando la barra oblicua en los enlaces, de esta forma:
</p>
<dl><dd>[[/Taller]]
<dl><dd>Lo cual formará un enlace dependiendo de la wiki en la cual lo visualice.
<dl><dd>Si está en Wikipedia, el enlace será -&gt; //es.wikipedia.org/wiki/Usuario:nombreusuario/Taller</dd>
<dd>Si está en Wikilibros, el enlace será -&gt; //es.wikibooks.org/wiki/Usuario:nombreusuario/Taller</dd></dl></dd></dl></dd></dl>
<p>Sin embargo, si la página "Taller" no está creada debidamente en Meta, el enlace se mostrará en rojo sin importar si está creada en la wiki desde la cual lo observe. Por lo que se recomienda usar un color personalizado para esta clase de enlaces.
</p>
<h4><span id="Limitaciones_t.C3.A9cnicas"></span><span class="mw-headline" id="Limitaciones_técnicas">Limitaciones técnicas</span></h4>
<p>Los enlaces que empiezan por dos barras oblicuas se rompen y no enlazan a ningún sitio. Así, vínculos como <tt>[[//]] o [[//Ejemplo]]</tt> simplemente no sirven. La solución es la misma que para los enlaces a subpáginas, con el inconveniente de que siempre son interpretados literalmente. Sin embargo, aunque no es recomendable, sí es posible crear subpáginas que incluyan una doble barra, pero para enlazarlas es necesario escribir su nombre completo.
</p><p>Si el nombre de la página debería comenzar de la misma forma que un <a href="/wiki/A:EN" class="mw-redirect" title="A:EN">espacio de nombres</a>, entonces no podría distinguirse como perteneciente al espacio principal y caería dentro de las <a href="/wiki/Wikipedia:RTT" class="mw-redirect" title="Wikipedia:RTT">restricciones a los títulos</a>. De esa forma, un vínculo como <tt>[[Ayuda: ¡sálvanos!]]</tt> conducirá siempre, aunque se le añadan dos puntos o se usen entidades HTML, al <a href="/wiki/Wikipedia:Espacio_de_nombres_Ayuda" title="Wikipedia:Espacio de nombres Ayuda">espacio Ayuda</a> y no a un artículo.
</p><p>Algunas palabras son interpretadas como una clave, llamada <b>prefijo</b>, que transforma el enlace interno en uno externo. Los prefijos pueden ser palabras completas o <a href="/wiki/Ap%C3%B3cope" title="Apócope">apócopes</a>, siempre seguido de dos puntos. Esta limitación es la más restrictiva, puesto que no es posible de ninguna forma enlazar a páginas que comiencen de la misma manera que un prefijo, no importa cuantos dos puntos se añadan ni si se usan entidades HTML. La razón es que <a href="/wiki/MediaWiki" title="MediaWiki">MediaWiki</a> los considera un <a href="/wiki/Ayuda:Enlace_interling%C3%BC%C3%ADstico" title="Ayuda:Enlace interlingüístico">enlace interlingüístico</a> mientras que si se añaden los dos puntos iniciales se interpretan como enlaces externos.
</p><p>Por ejemplo <tt>[[en:]]</tt> vincula la página en que se incluye con la portada de la <a href="/wiki/Wikipedia_en_ingl%C3%A9s" title="Wikipedia en inglés">Wikipedia en inglés</a>, mientras que <tt>[[:en:]] y [[&amp;#58;en:]]</tt> son enlaces externos que redirige a esa misma página.
</p><p>En <a href="/wiki/Especial:Enlaces_interlig%C3%BC%C3%ADsticos" title="Especial:Enlaces interligüísticos">Especial:Enlaces interligüísticos</a> tienes una lista completa de los prefijos disponibles.
</p>

<h3><span class="mw-headline" id="Enlaces_externos">Enlaces externos</span></h3>
<div class="thumb tright"><div class="thumbinner" style="width:302px;"><a href="/wiki/Archivo:Tutorial_de_edici%C3%B3n,_enlaces_externos.png" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/8d/Tutorial_de_edici%C3%B3n%2C_enlaces_externos.png/300px-Tutorial_de_edici%C3%B3n%2C_enlaces_externos.png" decoding="async" width="300" height="180" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/8/8d/Tutorial_de_edici%C3%B3n%2C_enlaces_externos.png 1.5x" data-file-width="400" data-file-height="240" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Archivo:Tutorial_de_edici%C3%B3n,_enlaces_externos.png" class="internal" title="Aumentar"></a></div>También existe un asistente para crear enlaces externos. Pulsa el botón <img alt="Enlace" src="//upload.wikimedia.org/wikipedia/commons/4/48/Vector_toolbar_insert_link_button.png" decoding="async" width="28" height="28" data-file-width="28" data-file-height="28" /> de la <a href="/wiki/Ayuda:Barra_de_herramientas" class="mw-redirect" title="Ayuda:Barra de herramientas">barra de herramientas</a> y selecciona la opción «A una página web externa». Escribe en «Título de página» la <a href="/wiki/Direcci%C3%B3n_web" class="mw-redirect" title="Dirección web">dirección web</a> o URL del sitio escogido (no olvides añadir al principio <code>http://</code>) y en «Título del enlace» el texto que hará de enlace.</div></div></div>
<p>Muchas veces es necesario enlazar con otros sitios web de Internet. Por ello, este tipo de enlaces se llaman <b>enlaces externos</b>.
</p><p>Ten en cuenta que en los artículos, los enlaces externos se colocan en las secciones "Referencias" y "Enlaces externos". En el cuerpo del artículo, <a href="/wiki/Wikipedia:Manual_de_estilo#Enlaces" title="Wikipedia:Manual de estilo">solo coloca enlaces dentro de Wikipedia en español</a>.
</p><p>El sistema detecta automáticamente los enlaces externos que empiecen por <code>http://</code>. Por ejemplo, al introducir lo siguiente (sin necesidad de dobles corchetes, ni de barra vertical):
</p>
<center><code>http://www.google.com</code></center>
<p>Obtendremos directamente un enlace externo:
</p>
<center><code><a rel="nofollow" class="external free" href="http://www.google.com">http://www.google.com</a></code></center>
<p>Si queremos que la dirección web quede oculta bajo un texto que nosotros elijamos, procederemos del siguiente modo:
</p>
<center><code>[http://www.google.com Página principal de Google]</code></center>
<p>Y resultará:
</p>
<center><code><a rel="nofollow" class="external text" href="http://www.google.com">Página principal de Google</a></code></center>
<p>También es posible crear un enlace externo de modo que aparezca como referencia numerada, encerrándola sin más entre corchetes simples (<code>[&#160;]</code>):
</p>
<center><code>[http://www.google.com]</code></center>
<p>Obtendremos:
</p>
<center><code><a rel="nofollow" class="external autonumber" href="http://www.google.com">[1]</a></code></center>
<h3><span class="mw-headline" id="Otros_enlaces">Otros enlaces</span></h3>
<div class="rellink noprint"> Para una lista detallada de los <b>prefijos</b>, véase <a href="/wiki/Especial:Enlaces_interlig%C3%BC%C3%ADsticos" title="Especial:Enlaces interligüísticos">Especial:Enlaces interligüísticos</a>.</div>
<p>Existen estas otras variantes a los enlaces internos y externos.
</p>
<ul><li>Títulos con paréntesis: <code>[[Júpiter (mitología)|]]</code> → <code><a href="/wiki/J%C3%BApiter_(mitolog%C3%ADa)" title="Júpiter (mitología)">Júpiter</a></code></li>
<li>Plurales:
<ul><li><code>[[rey]]es</code> → <code><a href="/wiki/Rey" class="mw-redirect" title="Rey">reyes</a></code></li>
<li><code>[[rey de Portugal|reyes de Portugal]]</code> → <code><a href="/wiki/Rey_de_Portugal" class="mw-redirect" title="Rey de Portugal">reyes de Portugal</a></code></li></ul></li>
<li>Imágenes y categorías: hay que precederlas de dos puntos (:).
<ul><li><code>[[:Categoría:Países]]</code> → <code><a href="/wiki/Categor%C3%ADa:Pa%C3%ADses" title="Categoría:Países">Categoría:Países</a></code></li>
<li><code>ver [[:Categoría:Países|todos los países]] del mundo</code> → <code>ver <a href="/wiki/Categor%C3%ADa:Pa%C3%ADses" title="Categoría:Países">todos los países</a> del mundo</code></li>
<li><code>[[:Archivo:Einstein.jpg]]</code> → <code><a href="/wiki/Archivo:Einstein.jpg" title="Archivo:Einstein.jpg">Archivo:Einstein.jpg</a></code></li>
<li><code>una imagen de [[:Archivo:Einstein.jpg|Albert Einstein]] pensativo</code> → <code>una imagen de <a href="/wiki/Archivo:Einstein.jpg" title="Archivo:Einstein.jpg">Albert Einstein</a> pensativo</code></li></ul></li>
<li>Títulos en un espacio de nombre distinto del principal: <code>[[Wikipedia:Café|]]</code> → <code><a href="/wiki/Wikipedia:Caf%C3%A9" title="Wikipedia:Café">Café</a></code></li>
<li>Subpáginas: establecen una jerarquía. Son útiles en páginas de usuario y discusión (para almacenar conversaciones antiguas).
<ul><li><code>[[/Subpágina]]</code>: enlaza a una subpágina de la actual. Es decir, otra forma de enlazar a <code>[[Wikipedia:Café/Archivo]]</code> desde «<a href="/wiki/Wikipedia:Caf%C3%A9" title="Wikipedia:Café">Wikipedia:Café</a>» es hacerlo así: <code>[[/Archivo|Archivo]]</code>.</li>
<li><code>[[../]]</code>: enlaza a la página superior de la actual.</li>
<li><code>[[../Hermana]]</code>: enlaza a una página en el mismo nivel de jerarquía. Es decir, desde <code>[[Wikipedia:Café/Archivo]]</code> podemos enlazar con <code>[[../Archivo 2004]]</code> a <code>[[Wikipedia:Café/Archivo 2004]]</code>.</li></ul></li>
<li>Secciones: permite enlazar con las secciones dentro de una misma página.
<ul><li><code>[[#Historia]]</code>: si el artículo tiene una sección llamada «Historia», el enlace nos llevará a ella.</li>
<li><code>[[#top]]</code>: envía al comienzo de la página.</li>
<li><code>[[#footer]]</code>: envía al final de la página.</li>
<li>Como con los demás enlaces internos, podemos modificar el texto del enlace: <code>[[#Historia|su historia]]</code> → <code><a href="#Historia">su historia</a></code></li></ul></li></ul>
<h2><span class="mw-headline" id="Formatos_de_texto">Formatos de texto</span></h2>
<p>Los <b>formatos de texto</b> facilitan la lectura y resaltan los puntos más importantes. Los principales son las <b>negritas</b> y las <i>cursivas</i>. También existen otras opciones adicionales.
</p><p>Los formatos de texto, sobre todo los subrayados, tachados, líneas horizontales, distintos tamaños, fuentes y colores, pueden provocar un efecto negativo en la legibilidad del texto. Por ello ten en cuenta que no conviene abusar de los formatos de texto, e intenta utilizarlos lo menos posible. Mira otros artículos para entender cuándo se suele utilizar negritas y cursivas.
</p>
<h3><span class="mw-headline" id="Negritas">Negritas</span></h3>
<p>Para convertir una palabra o frase en <b>negritas</b> debes rodearlo de <b>tres comillas simples</b> (<code>'''</code>), escribiéndolas a mano o presionando el botón <img alt="Negrita" src="//upload.wikimedia.org/wikipedia/commons/c/ca/Vector_toolbar_bold_N_button.png" decoding="async" width="28" height="28" data-file-width="28" data-file-height="28" /> de la <a href="/wiki/Ayuda:Barra_de_herramientas" class="mw-redirect" title="Ayuda:Barra de herramientas">barra de herramientas</a>. El código siguiente:
</p>
<center><code>...el texto en negritas '''resalta mucho''' sobre todo lo demás...</code></center>
<p>Genera esto:
</p>
<center><code>...el texto en negritas <b>resalta mucho</b> sobre todo lo demás...</code></center>
<h3><span class="mw-headline" id="Cursivas">Cursivas</span></h3>
<p>Para convertir una palabra o frase en <i>cursivas</i> debes rodearlo de <b>dos comillas simples</b> (<code>''</code>), escribiéndolas a mano o presionando el botón <img alt="Negritas" src="//upload.wikimedia.org/wikipedia/commons/5/50/Vector_toolbar_italic_C_button.png" decoding="async" width="28" height="28" data-file-width="28" data-file-height="28" /> de la <a href="/wiki/Ayuda:Barra_de_herramientas" class="mw-redirect" title="Ayuda:Barra de herramientas">barra de herramientas</a>. El código siguiente:
</p>
<center><code>...el texto en cursiva ''resalta un poco menos'' que en negrita...</code></center>
<p>Genera esto:
</p>
<center><code>...el texto en cursiva <i>resalta un poco menos</i> que en negrita...</code></center>
<h3><span class="mw-headline" id="Otras_opciones_de_formato">Otras opciones de formato</span></h3>
<p>Además de las negritas y cursivas existen <b>otros formatos de texto</b>:
</p>
<h4><span class="mw-headline" id="Subrayado">Subrayado</span></h4>
<p><code>...podemos subrayar &lt;u&gt;una parte&lt;/u&gt; de la oración...</code> → ...podemos subrayar <u>una parte</u> de la oración...
</p>
<h4><span class="mw-headline" id="Tachado">Tachado</span></h4>
<p><code>...incluso tachar &lt;s&gt;un fragmento&lt;/s&gt; de otra...</code> → ...incluso tachar <s>un fragmento</s> de otra...
</p>
<h4><span id="Sangr.C3.ADa"></span><span class="mw-headline" id="Sangría">Sangría</span></h4>
<p>La forma más simple de poner sangría al texto es escribir «dos puntos» (<code>:</code>) al inicio de la línea. Cuantos más signos de dos puntos pongas, más sangría tendrá la línea.
</p>
<dl><dd><code>Esta línea está sangrada a nivel 1</code>
<dl><dd><code>Esta lo está un poco más, a nivel 2 (se ponen dos caracteres de <i>dos puntos</i>&#160;:: al inicio)</code>
<dl><dd><code>Nivel 3. ¿Quién sabe hasta dónde podríamos llegar? (se ponen tres signos de <i>dos puntos</i>&#160;::: al inicio)</code></dd></dl></dd></dl></dd></dl>
<p>Cuando hay mucha sangría se usa la plantilla <code>{{<a href="/wiki/Plantilla:Quito_sangr%C3%ADa" class="mw-redirect" title="Plantilla:Quito sangría">quito sangría</a>}}</code> y así se empieza desde el principio de la frase, y entre la nueva frase y la vieja aparece una línea como la siguiente. Hay que poner {{quito sangría|número de puntos}}, en <code>número de puntos</code> sustituirlo por la misma cantidad de puntos que la última frase, también se puede poner un número entero. Quedando así {{quito sangría|:::}} o {{quito sangría|3}} <br />
<span class="outdent-template" style="font-family:DejaVu Sans, sans-serif;"><span style="color:#808080">┌──────┘</span><br /></span>
</p>
<h4><span id="Tipograf.C3.ADa_monoespaciada"></span><span class="mw-headline" id="Tipografía_monoespaciada">Tipografía monoespaciada</span></h4>
<p>Para escribir un texto con una <a href="/wiki/Tipograf%C3%ADa" title="Tipografía">tipografía</a> de anchura fija de carácter, como «Courier», escribe uno o más espacios al principio de línea:
</p>
<pre>En otras palabras, si empiezas un párrafo con espacios, esa línea se indentará
y a la vez aparecerá con una fuente monoespaciada.
</pre>
<h4><span class="mw-headline" id="Coloreado_de_sintaxis">Coloreado de sintaxis</span></h4>
<p>Si quieres insertar fragmentos de código fuente puedes hacer su lectura más sencilla si usas el coloreado de sintaxis:
</p>
<pre>&lt;source lang=C&gt;
int main()
{
    printf("hola");
}
&lt;/source&gt;
</pre>
<p>Se verá así:
</p>
<div class="mw-highlight mw-content-ltr" dir="ltr"><pre><span></span><span class="kt">int</span> <span class="nf">main</span><span class="p">()</span>
<span class="p">{</span>
    <span class="n">printf</span><span class="p">(</span><span class="s">&quot;hola&quot;</span><span class="p">);</span>
<span class="p">}</span>
</pre></div>
<h4><span class="mw-headline" id="Tipos_de_letra">Tipos de letra</span></h4>
<p>Existen etiquetas para cambiar el tipo de fuente, de esta manera resalta sobre el texto normal.
</p>
<ul><li><code>&lt;tt&gt;&lt;/tt&gt;</code>: fuente monoespacio. Ejemplo: <code>&lt;tt&gt;monoespacio&lt;/tt&gt;</code> → <tt>monoespacio</tt>.</li>
<li><code>&lt;code&gt;&lt;/code&gt;</code>: resalta el texto levemente mediante un color de fondo gris. Ejemplo: <code>&lt;code&gt;mi código&lt;/code&gt;</code> → <code>mi código</code>.</li>
<li><code>&lt;pre&gt;&lt;/pre&gt;</code>: similar a <code>&lt;code&gt;&lt;/code&gt;</code> pero preserva los saltos de línea.</li>
<li><code>&lt;span&gt;&lt;/span&gt;</code>: para escribir en una fuente que desees. Ejemplo para escribir usando <a href="/wiki/Times_New_Roman" title="Times New Roman">Times New Roman</a>: <code>&lt;span style="font-family: Times New Roman"&gt;Aquí el contenido&lt;/span&gt;</code> → <code><span style="font-family: Times New Roman">Aquí el contenido</span></code>.</li>
<li><code>&lt;nowiki&gt;&lt;/nowiki&gt;</code>: su utilidad es la de poder insertar código wiki sin que sea interpretado. En el siguiente ejemplo no se activa el enlace: <code>&lt;nowiki&gt;[[enlace]]&lt;/nowiki&gt;</code> → <code>[[enlace]]</code>. Una forma cómoda de usar <code>&lt;nowiki&gt;&lt;/nowiki&gt;</code> es seleccionar el código que quieras mostrar y pulsar el botón <img alt="Sin formato wiki" src="//upload.wikimedia.org/wikipedia/commons/8/87/Vector_toolbar_no_wiki_formatting_button.png" decoding="async" width="28" height="28" data-file-width="28" data-file-height="28" />.</li></ul>

<h4><span id="Super.C3.ADndices_y_sub.C3.ADndices"></span><span class="mw-headline" id="Superíndices_y_subíndices">Superíndices y subíndices</span></h4>
<p>Con las etiquetas <code>&lt;sup&gt;&lt;/sup&gt;</code> y <code>&lt;sub&gt;&lt;/sub&gt;</code> respectivamente, insertando el texto dentro de ellas. Ejemplo: <code>&lt;sup&gt;texto arriba&lt;/sup&gt;</code> → <sup>texto arriba</sup> y <code>&lt;sub&gt;texto abajo&lt;/sub&gt;</code> → <sub>texto abajo</sub>. Recuerda que tienes a tu disposición los botones <img alt="Subíndice" src="//upload.wikimedia.org/wikipedia/commons/d/d0/Vector_toolbar_subscript_button.png" decoding="async" width="28" height="28" data-file-width="28" data-file-height="28" /> <img alt="Superíndice" src="//upload.wikimedia.org/wikipedia/commons/c/c8/Vector_toolbar_superscript_button.png" decoding="async" width="28" height="28" data-file-width="28" data-file-height="28" /> (subíndice / superíndice).
</p>
<h4><span id="Tama.C3.B1o_del_texto"></span><span class="mw-headline" id="Tamaño_del_texto">Tamaño del texto</span></h4>
<p>Puedes modificarlo metiendo el texto entre las etiquetas <code>&lt;small&gt;&lt;/small&gt;</code> para reducirlo o entre <code>&lt;big&gt;&lt;/big&gt;</code> para agrandarlo; usa los botones <img alt="Texto grande" src="//upload.wikimedia.org/wikipedia/commons/7/77/Vector_toolbar_big_text_button.png" decoding="async" width="28" height="28" data-file-width="28" data-file-height="28" /> <img alt="Texto pequeño" src="//upload.wikimedia.org/wikipedia/commons/3/3c/Vector_toolbar_small_text_button.png" decoding="async" width="28" height="28" data-file-width="28" data-file-height="28" /> (agrandar/reducir tamaño del texto) para hacerlo sin mayores complicaciones. Ejemplo: <code>&lt;small&gt;texto pequeño&lt;/small&gt;</code> → <small>texto pequeño</small> y <code>&lt;big&gt;texto grande&lt;/big&gt;</code> → <big>texto grande</big>.
</p>

<p>También puedes controlar el <font color="#00FF00"><b>c</b></font><font color="#0000FF"><b>o</b></font><font color="#00FFFF"><b>l</b></font><font color="#FF00FF"><b>o</b></font><font color="#FF8000"><b>r</b></font>. Por ejemplo: <code>&lt;span style="color: #FF00FF"&gt;este texto es rosa&lt;/span&gt;</code> → <span style="color: #FF00FF">este texto es rosa</span>. Para ver un listado de los colores que puedes usar, mira <a href="/wiki/Colores_HTML" class="mw-redirect" title="Colores HTML">Colores HTML</a>.

<h4><span class="mw-headline" id="Comentarios">Comentarios</span></h4>
<p>Es posible colocar comentarios en la edición. En modo edición al insertar líneas comentadas, estas solo se verán al editar y no aparecerán en el artículo. Los comentarios son particularmente útiles cuando se traduce de una wiki de un idioma a otro, o cuando no se está seguro de hacer la traducción adecuada de un trozo de texto.
</p><p>Por ejemplo <code>&lt;!--mi comentario--&gt;</code>
</p>
<h2><span class="mw-headline" id="Secciones">Secciones</span></h2>
<p>Cuando una página tiene bastante texto y la separación en párrafos no es suficiente, se crean <b>secciones</b> para estructurar el texto. Por ejemplo, el artículo <a href="/wiki/Historia_de_Francia" title="Historia de Francia">Historia de Francia</a> tiene una sección por cada época. A su vez, cada sección puede tener subsecciones.
</p><p>El título de la sección se sitúa entre dos o más signos de igual <code>== Sección ==</code>. Los espacios entre los signos de igual y el título no se tienen en cuenta. Es lo mismo poner <code>==Filmografía==</code> que <code>== Filmografía ==</code>. Aunque sí es obligatorio pegar el primer signo de igual al margen izquierdo.
</p><p>El número de signos de igual define el nivel de jerarquía de la sección del siguiente modo:
</p>
<dl><dd><code>== Sección 1 ==</code><br /></dd>
<dd><code>=== Sección 1.1 ===</code><br /></dd>
<dd><code>==== Sección 1.1.1 ====</code><br /></dd>
<dd><code>==== Sección 1.1.2 ====</code><br /></dd>
<dd><code>=== Sección 1.2 ===</code><br /></dd>
<dd><code>== Sección 2 ==</code><br /></dd>
<dd><code>== Sección 3 ==</code><br /></dd></dl>
<p>También puedes crear secciones utilizando la <a href="/wiki/Ayuda:Barra_de_herramientas" class="mw-redirect" title="Ayuda:Barra de herramientas">barra de herramientas</a> pulsando el menú desplegable «Encabezado».
</p>
<div class="center"><div class="floatnone"><a href="/wiki/Archivo:Vector_toolbar_full-es.png" class="image" title="Barra de herramientas"><img alt="Barra de herramientas" src="//upload.wikimedia.org/wikipedia/commons/4/49/Vector_toolbar_full-es.png" decoding="async" width="700" height="67" data-file-width="700" data-file-height="67" /></a></div></div>
<p>Junto a las secciones aparece un enlace [<font color="blue">editar</font>], que permite modificar solo esa sección. Esto facilita la edición de artículos largos.
</p><p>En el caso de que, por tener la página un diseño especial, no convenga que aparezcan esos enlaces, puedes eliminarlos poniendo <code>__NOEDITSECTION__</code>.
</p>
<h2><span class="mw-headline" id="Archivos">Archivos</span></h2>
<p>En Wikipedia en español únicamente pueden utilizarse los <b>archivos</b> disponibles en el repositorio multimedia <a href="/wiki/Wikimedia_Commons" title="Wikimedia Commons">Wikimedia Commons</a>. Por ello, si quieres insertar una imagen o recurso en un artículo, antes debes subirlos a Wikimedia Commons (ir al <b><a href="https://commons.wikimedia.org/wiki/Special:UploadWizard" class="extiw" title="commons:Special:UploadWizard">formulario de subida</a></b>).
</p>
<center><div style="width: 90%;border: 1px solid gray;padding: 5px;background-color: #FFCCCC;">Recuerda que <b>solo se permiten archivos con <a href="https://commons.wikimedia.org/wiki/Commons:Sobre_las_licencias" class="extiw" title="commons:Commons:Sobre las licencias">licencias libres</a></b> (como <a href="/wiki/Creative_Commons" title="Creative Commons">Creative Commons</a> en sus variantes CC-BY y CC-BY-SA, <a href="/wiki/Licencia_Arte_Libre" title="Licencia Arte Libre">Licencia Arte Libre</a> o <a href="/wiki/Dominio_p%C3%BAblico" title="Dominio público">dominio público</a>).
<p>La mayoría de las imágenes y recursos de otros sitios web de Internet no son libres (si no indican nada, se consideran «Todos los derechos reservados»), por lo que no sirven para Wikipedia (y si los subes, serán borrados).
</p>
Las imágenes, audio y vídeo creados por ti sirven, siempre que decidas publicarlos con una de las licencias libres mencionadas anteriormente.</div></center>
<p>Una vez que has subido el archivo a Wikimedia Commons, tienes que insertar un código en el artículo de Wikipedia para que se muestre. Este código se compone de dos partes fundamentales: el <b>nombre del archivo</b> y las <b>opciones</b> de visualización.
</p>
<center><code>[[Archivo:<i>Nombre del archivo</i>|<i>opciones</i>]]</code></center>
<p>Puedes hacerlo con el botón <img alt="Archivo empotrado" src="//upload.wikimedia.org/wikipedia/commons/d/d8/Vector_toolbar_insert_image_button.png" decoding="async" width="28" height="28" data-file-width="28" data-file-height="28" /> que aparece en la <a href="/wiki/Ayuda:Barra_de_herramientas" class="mw-redirect" title="Ayuda:Barra de herramientas">barra de herramientas</a>. Escribe y selecciona el nombre del archivo y el texto alternativo —separados por una barra «|»— y pulsa el botón. Escribirá <tt><b>[[Archivo:</b></tt> adelante y <tt><b>]]</b></tt> detrás.
</p><p>El <b>nombre del archivo</b> siempre debe tener una extensión (la mayoría de las imágenes son <a href="/wiki/JPG" class="mw-redirect" title="JPG">JPG/JPEG</a>, <a href="/wiki/PNG" class="mw-redirect" title="PNG">PNG</a> o <a href="/wiki/Scalable_Vector_Graphics" class="mw-redirect" title="Scalable Vector Graphics">SVG</a>; para audio o vídeo puedes usar <a href="/wiki/OGG" class="mw-redirect" title="OGG">OGG</a> y para documentos <a href="/wiki/PDF" title="PDF">PDF</a> o <a href="/wiki/DjVu" title="DjVu">DjVu</a>). Debes tener cuidado con las mayúsculas y minúsculas. Por ejemplo, si subes una imagen con el nombre «<i>Mi Imagen.jpg</i>» (atención a la «<i>i</i>» mayúscula) pero luego no escribes su nombre exactamente igual, no funcionará y dicha imagen no se verá (saldrá un enlace rojo indicando que la imagen no existe). Sin embargo, da igual escribir espacios o guiones bajos «<code>_</code>», los nombres «<i>Mi Imagen.jpg</i>» y «<i>Mi_Imagen.jpg</i>» son equivalentes.
</p><p>Algunas de las <b>opciones</b> de visualización avanzadas al insertar archivos son las siguientes:
</p>
<ul><li><b>Formato</b>: <code>miniatura</code> o <code>thumb</code> (miniatura y con recuadro), <code>frame</code> (marco) o <code>border</code> (con un borde gris). Es recomendable utilizar <code>miniatura</code>.</li>
<li><b>Alineación</b>: posición del archivo en la pantalla (se puede indicar en español o inglés). Puede ser: <code>left</code> o <code>izquierda</code>, <code>center</code> o <code>centro</code>, <code>right</code> o <code>derecha</code>. Lo común es en la derecha.</li>
<li><b>Tamaño</b>: se indica en <a href="/wiki/P%C3%ADxeles" class="mw-redirect" title="Píxeles">píxeles</a> de ancho. Por ejemplo <code>200px</code> ó <code>300px</code>. Si no existe la necesidad de usarlo más grande, lo mejor es algo menor o igual a <code>220px</code>. Recuerda que aún hay gente con <a href="/wiki/Resoluciones_de_pantalla" class="mw-redirect" title="Resoluciones de pantalla">resoluciones de pantalla</a> pequeñas. También puede especificarse el tamaño del alto de la imagen colocando la letra <code>x</code> antes del valor en píxeles, como por ejemplo <code>x220px</code>. Ten en cuenta que no es posible especificar ambos; en esos casos se da prioridad al ancho.</li>
<li><b>Descripción</b>: es el texto que explica el contenido del archivo (en el caso de las imágenes es el pie de foto). Puede ser todo lo detallado que se quiera.</li>
<li><b>Texto alternativo</b>: se indica mediante el código <code>alt=</code>. Allí puedes añadir un texto narrativo de lo que se ve. Es recomendable ponerlo siempre, de esta forma, los <a href="/wiki/Invidentes" class="mw-redirect" title="Invidentes">invidentes</a> que utilizan <a href="/wiki/Lectores_de_pantalla" class="mw-redirect" title="Lectores de pantalla">lectores de pantalla</a> pueden hacerse una idea de qué se muestra en el archivo.</li>
<li><b>Enlace</b>: se indica mediante el código <code>link=</code>. Añadir el nombre de una página de Wikipedia o una dirección web allí, hará que la imagen lleve hacia esa página al hacer clic sobre ella, en lugar de la <a href="/wiki/Ayuda:P%C3%A1gina_de_descripci%C3%B3n_de_las_im%C3%A1genes" title="Ayuda:Página de descripción de las imágenes">página de descripción</a> por defecto.</li></ul>
<p>A un archivo se le puede pasar ninguna opción, una, varias o todas (y en cualquier orden, la única obligación es poner el nombre del archivo al principio y separar las opciones con barras verticales «|»).
</p>

Puedes ver más información sobre la imagen pulsando en ella.</div></div></div>
<p>A continuación se detallan algunos ejemplos para insertar <b>imágenes</b> en artículos:
</p>
<ul><li><code>[[Archivo:Mi imagen.jpg]]</code>: no hemos indicado nada. Quedará a tamaño máximo y flotando a la izquierda.</li>
<li><code>[[Archivo:Mi imagen.jpg|thumb]]</code>: saldrá con un recuadro alrededor. Por defecto, debido al <code>thumb</code>, la posición será a la derecha y con 180 píxeles de tamaño (a menos que el lector lo haya cambiado en sus preferencias).</li>
<li><code>[[Archivo:Mi imagen.jpg|border|100px]]</code>: con un borde gris alrededor y a 100 píxeles.</li>
<li><code>[[Archivo:Mi imagen.jpg|thumb|left]]</code>: con recuadro y a la izquierda. El tamaño será de 180 píxeles (indicados por el <code>thumb</code> por defecto).</li>
<li><code>[[Archivo:Mi imagen.jpg|thumb|right|350px]]</code>: con recuadro, a la derecha y a 350 píxeles.</li>
<li><code>[[Archivo:Mi imagen.jpg|thumb|right|200px|La descripción de mi imagen.]]</code>: con recuadro, a la derecha, a 200 píxeles y con una descripción útil.</li></ul>
<p>Lo más recomendable es utilizar <code>[[Archivo:Mi imagen.jpg|thumb|La descripción de mi imagen.]]</code>. Debido al <code>thumb</code> se mostrará con un recuadro, a la derecha y a 180 píxeles. Si el lector ha elegido otros valores en sus preferencias, se mostrará con ellos, por lo que estamos dejándole esa libertad de elección (lo cual es bueno).
</p>
<h3><span id="Audio_y_v.C3.ADdeo"></span><span class="mw-headline" id="Audio_y_vídeo">Audio y vídeo</span></h3>
<div class="rellink noprint"> Para obtener ayuda sobre cómo reproducir archivos multimedia, véase <a href="/wiki/Ayuda:Multimedia" title="Ayuda:Multimedia">Ayuda:Multimedia</a>.</div>
<p>En Wikipedia se utilizan únicamente los <a href="/wiki/Formato_abierto" title="Formato abierto">formatos abiertos</a> <a href="/wiki/Ogg" title="Ogg">Ogg</a> y <a href="/wiki/WebM" title="WebM">WebM</a> para recursos multimedia. Por ende, los archivos de sonido y vídeo que encuentres, usarán los códecs libres <a href="/wiki/Vorbis" title="Vorbis">Vorbis</a>, <a href="/wiki/Theora" title="Theora">Theora</a> y <a href="/wiki/WebM" title="WebM">WebM</a>, los cuales son similares en su funcionamiento a formatos más conocidos como <a href="/wiki/MP3" title="MP3">MP3</a> y <a href="/wiki/MPEG-2" title="MPEG-2">MPEG-2</a>/<a href="/wiki/MPEG-4" title="MPEG-4">4</a>. Dichos formatos (como el caso de MP3) están protegidos por patentes y derechos de autor por lo que no se permiten aquí.
</p><p>La manera más fácil de añadir un recurso multimedia (especialmente audio) es usando la plantilla <code>{{<a href="/wiki/Plantilla:Multimedia" title="Plantilla:Multimedia">Multimedia</a>}}</code>; haz clic en su enlace para ver una descripción detallada de cómo utilizarla. A continuación se resumen sus parámetros más comunes:
</p>
<dl><dd><code>{{Multimedia|archivo=<i>NOMBRE</i>.ogg|título=<i>TÍTULO</i>|descripción=<i>DESCRIPCIÓN</i>}}</code></dd></dl>

<p>Existen muchas otras plantillas de audio, utilizadas principalmente para demostrar la pronunciación de términos en idiomas extranjeros, como las plantillas <code>{{<a href="/wiki/Plantilla:Audio" title="Plantilla:Audio">audio</a>}}</code>/<code>{{<a href="/wiki/Plantilla:Audio-IPA" class="mw-redirect" title="Plantilla:Audio-IPA">audio-IPA</a>}}</code>. <h3><span id="Notaci.C3.B3n_musical"></span><span class="mw-headline" id="Notación_musical">Notación musical</span></h3>
<p>También se pueden insertar fragmentos de notación musical y secuencias de audio <a href="/wiki/MIDI" title="MIDI">MIDI</a> en Wikipedia usando las etiquetas <code>&lt;score&gt;&lt;/score&gt;</code> mediante el software <a href="/wiki/LilyPond" title="LilyPond">LilyPond</a> incorporado en el código de <a href="/wiki/MediaWiki" title="MediaWiki">MediaWiki</a> (<a class="external text" href="https://www.mediawiki.org/wiki/Extension:Score/es">Score Extension</a>).
</p><p>Por ejemplo, el siguiente fragmento de código...
</p>
<pre style="overflow:auto">&lt;score&gt;\relative c' { f d f a d f e d cis a cis e a g f e }&lt;/score&gt;
</pre>
<p>...se convierte en la siguiente notación musical.
</p>
<div class="mw-ext-score" data-midi="//upload.wikimedia.org/score/r/5/r5jd72zh3jw655tykroq3sepak6ea0a/r5jd72zh.midi" data-source="//upload.wikimedia.org/score/r/5/r5jd72zh3jw655tykroq3sepak6ea0a/r5jd72zh.ly"><img src="//upload.wikimedia.org/score/r/5/r5jd72zh3jw655tykroq3sepak6ea0a/r5jd72zh.png" width="434" height="52" alt="\relative c&#39; { f d f a d f e d cis a cis e a g f e }" /></div>
<p>El anterior ejemplo es una versión simplificada del <a href="/w/index.php?title=Solfeggietto&amp;action=edit&amp;redlink=1" class="new" title="Solfeggietto (aún no redactado)">Solfeggietto</a> en do menor de <a href="/wiki/Carl_Philipp_Emanuel_Bach" title="Carl Philipp Emanuel Bach">Carl Philipp Emanuel Bach</a>.
</p><p>Si se quiere generar un archivo de audio que coincida con el ejemplo, solo se debe agregar el parámetro <code>|vorbis=1</code> tras la etiqueta de apertura <code>&lt;score&gt;</code> y se hará automáticamente:
</p>
<pre style="overflow:auto">&lt;score vorbis="1"&gt;\relative c' { f d f a d f e d cis a cis e a g f e }&lt;/score&gt;
</pre>
<div class="mw-ext-score" data-midi="//upload.wikimedia.org/score/r/5/r5jd72zh3jw655tykroq3sepak6ea0a/r5jd72zh.midi" data-source="//upload.wikimedia.org/score/r/5/r5jd72zh3jw655tykroq3sepak6ea0a/r5jd72zh.ly"><img src="//upload.wikimedia.org/score/r/5/r5jd72zh3jw655tykroq3sepak6ea0a/r5jd72zh.png" width="434" height="52" alt="\relative c&#39; { f d f a d f e d cis a cis e a g f e }" /><div class="mediaContainer" style="width:300px"><audio id="mwe_player_0" controls="" preload="none" style="width:300px" class="kskin" data-disablecontrols="options,timedText" data-durationhint="9.6029024943311"><source src="//upload.wikimedia.org/score/r/5/r5jd72zh3jw655tykroq3sepak6ea0a/r5jd72zh.ogg" type="application/ogg" /></audio></div></div>
<p>Además de ejemplos cortos, es posible insertar notación musical o partituras más complejas (incluyendo letras de voz) usando códigos LilyPond. Puedes visitar <a rel="nofollow" class="external text" href="http://lilypond.org/doc/v2.14/Documentation/learning-big-page#common-notation">este enlace</a> para obtener más información sobre la notación corriente de LilyPond.

<h2><span class="mw-headline" id="Listas">Listas</span></h2>
<div class="noprint AP rellink"><span style="font-size:88%">Artículo principal:</span>&#32;<i><a href="/wiki/Ayuda:Listas" title="Ayuda:Listas"> Ayuda:Listas</a></i></div>
<p>Las <b>listas</b> permiten organizar visualmente la información.
</p>
<h3><span class="mw-headline" id="Numeradas">Numeradas</span></h3>
<p>Las <b>listas numeradas</b> se crean utilizando el símbolo <code>#</code> pegado al margen izquierdo. Para añadir una fácilmente, utiliza el botón <img alt="Lista numerada" src="//upload.wikimedia.org/wikipedia/commons/3/30/Vector_toolbar_numbered_list_button.png" decoding="async" width="28" height="28" data-file-width="28" data-file-height="28" /> situado en la sección «Avanzado» de la <a href="/wiki/Ayuda:Barra_de_herramientas" class="mw-redirect" title="Ayuda:Barra de herramientas">barra de herramientas</a>.
</p><p>Las listas numeradas se utilizan cuando se necesita indicar la posición de cada elemento, por ejemplo al describir una secuencia de pasos.
</p><p>Este es un ejemplo de una lista numerada:
</p><p><code># Ítem uno</code><br />
<code># Ítem dos</code><br />
<code>## Se pueden anidar poniendo dos o más # seguidos</code><br />
<code>## Otra</code><br />
<code>### Todavía más...</code><br />
<code># Ítem tres</code><br />
</p><p>Que genera lo siguiente:
</p>
<ol><li>Ítem uno</li>
<li>Ítem dos
<ol><li>Se pueden anidar poniendo dos o más <code>#</code> seguidos</li>
<li>Otra
<ol><li>Todavía más...</li></ol></li></ol></li>
<li>Ítem tres</li></ol>
<h3><span class="mw-headline" id="No_numeradas">No numeradas</span></h3>
<p>Las <b>listas no numeradas</b> (o viñeteadas) se crean con el signo de asterisco <code>*</code>. Si no quieres escribirlos a mano, utiliza el botón <img alt="Lista viñeteada" src="//upload.wikimedia.org/wikipedia/commons/2/25/Vector_toolbar_bulleted_list_button.png" decoding="async" width="28" height="28" data-file-width="28" data-file-height="28" /> de la <a href="/wiki/Ayuda:Barra_de_herramientas" class="mw-redirect" title="Ayuda:Barra de herramientas">barra de herramientas</a>.
</p><p>Las listas no numeradas son las más comunes, y se utilizan cuando el orden de los puntos no es relevante.
</p><p>Con el siguiente código para listas no numeradas:
</p><p><code>* Un punto</code><br />
<code>* Otro punto</code><br />
<code>** Podemos tener también sublistas (usando varios * seguidos)</code><br />
<code>*** Y seguir, y seguir...</code><br />
<code>* Último punto</code><br />
</p><p>Tendremos esto:
</p>
<ul><li>Un punto</li>
<li>Otro punto
<ul><li>Podemos tener también sublistas (usando varios <code>*</code> seguidos)
<ul><li>Y seguir, y seguir...</li></ul></li></ul></li>
<li>Último punto</li></ul>
<h3><span class="mw-headline" id="Otras_listas">Otras listas</span></h3>
<p>Las <b>listas de definición</b> se generan empezando la línea con el signo de punto y coma <code>;</code>. El resto de elementos de la lista se generan poniendo el signo de dos puntos <code>:</code> ya sea en la misma línea del título de la lista (si es de solo un elemento) o en una nueva línea. Si necesitas incluir los dos puntos como signo de puntuación dentro del título de la lista, tendrás que sustituirlo por su codificación en HTML (<code>&amp;#58;</code>).
</p><p>Las listas de definición se utilizan poco en artículos. En cambio, se utilizan para realizar comentarios en las páginas de discusión.
</p><p>Si introducimos este código:
</p><p><code>; Este es el título de la lista de definición : Este es un elemento.</code><br />
<code>: este es otro elemento.</code><br />
<code>:: uno más.</code><br />
<code>: y este otro.</code><br />
</p><p>Veremos esta salida:
</p>
<dl><dt>Este es el título de la lista de definición&#160;</dt>
<dd>Este es un elemento.</dd>
<dd>este es otro elemento.
<dl><dd>uno más.</dd></dl></dd>
<dd>y este otro.</dd></dl>

<h3><span id=".C3.8Dndice"></span><span class="mw-headline" id="Índice">Índice</span></h3>
<p>En esta misma página hay al principio un <b>índice</b>, pero si intentas editarlo, verás que no puedes. ¿Por qué? Los índices se generan automáticamente cuando hay más de tres <a href="#Secciones">secciones</a> en la página. Las secciones se numeran según la jerarquía. Se da el número 1 a la primera sección que haya, independientemente del nivel que tenga y será considerada la raíz del árbol jerárquico. Por eso esta página, aunque empieza en un nivel 2, en el índice aparece el título como 1 y no como 1.1.
</p><p>Para un control más preciso de los índices, hay una serie de órdenes que puedes poner en el texto del artículo. Se recomienda usarlas solo cuando el índice puede crear algún problema (por ejemplo, en listas de artículos o diseños especiales):
</p>
<ul><li><code>__FORCETOC__</code> hace aparecer el índice, aunque no haya suficientes secciones. Debes ponerlo al principio o al final del artículo.</li>
<li><code>__TOC__</code> hace aparecer el índice en la parte de la página donde lo pongas.</li>
<li><code>__NOTOC__</code> hace que el índice no aparezca, por muchas secciones que tenga el artículo. Ponlo al principio o al final del artículo.</li></ul>
<h2><span id="F.C3.B3rmulas"></span><span class="mw-headline" id="Fórmulas">Fórmulas</span></h2>
<div class="noprint AP rellink"><span style="font-size:88%">Artículo principal:</span>&#32;<i><a href="/wiki/Ayuda:F%C3%B3rmulas" class="mw-redirect" title="Ayuda:Fórmulas"> Ayuda:Fórmulas</a></i></div>
<p>En algunos artículos, por ejemplo los de matemáticos, es necesario incluir <b>fórmulas</b> más o menos complejas. Para ello se utiliza <a href="/wiki/TeX" title="TeX">TeX</a>. Por ejemplo, con el código <code>&lt;math&gt;x=\frac{-b\pm\sqrt{b^2-4ac}}{2a}&lt;/math&gt;</code>, se obtiene:
</p>

<h4><span class="mw-headline" id="Firmar_un_comentario">Firmar un comentario</span></h4>
<table style="border: 1px solid #aaa; background: #fff; margin: 0 0 1em 1em; padding: 3px; float: right; width: 67px; font-size: 90%;" class="noprint plainlinks">

<tbody><tr>
<th id="shortcut" style="text-align:center; vertical-align:top; border: none; background: transparent;"><a href="/wiki/Ayuda:Atajos" title="Ayuda:Atajos">Atajo</a><hr /><span class="sin-enlace" style="display:none">A:FC</span><a class="external text" href="https://es.wikipedia.org/w/index.php?title=A:FC&amp;redirect=no">A:FC</a>
</th></tr></tbody></table>
<div class="VT rellink"><span style="font-size:88%">Véase también:</span> <i><a href="/wiki/Wikipedia:Firma_de_usuario" title="Wikipedia:Firma de usuario">Wikipedia:Firma de usuario</a></i></div>
<p>En muchos sitios de Wikipedia, como las <a href="/wiki/Wikipedia:P%C3%A1ginas_de_discusi%C3%B3n" title="Wikipedia:Páginas de discusión">páginas de discusión</a>, el <a href="/wiki/Wikipedia:Caf%C3%A9" title="Wikipedia:Café">Café</a>, y en los <a href="/wiki/Wikipedia:Tabl%C3%B3n_de_anuncios_de_los_bibliotecarios/Portal2" title="Wikipedia:Tablón de anuncios de los bibliotecarios/Portal2">tablones de bibliotecario</a> como la lista de vandalizadas, cuando escribes un mensaje o comentario, se te pide que lo firmes. Para ello no tienes que hacerlo «a mano» sino poniendo cuatro <a href="/wiki/Virgulilla" title="Virgulilla">virgulillas</a> ~~~~. Este símbolo se corresponde con el <a href="/wiki/ASCII" title="ASCII">ASCII</a> 126 y puede consultar <a href="/wiki/Virgulilla#Escritura_con_teclado_de_ordenador" title="Virgulilla">aquí</a> como escribirlas en su teclado. Debido al uso frecuente del proceso de firma, Wikipedia permite que se firme, de forma más accesible, pulsando el botón <a href="/wiki/Archivo:OOUI_JS_signature_icon_LTR.svg" class="image" title="Firma y fecha"><img alt="Firma y fecha" src="//upload.wikimedia.org/wikipedia/commons/thumb/1/1d/OOUI_JS_signature_icon_LTR.svg/22px-OOUI_JS_signature_icon_LTR.svg.png" decoding="async" width="22" height="22" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/1/1d/OOUI_JS_signature_icon_LTR.svg/33px-OOUI_JS_signature_icon_LTR.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/1/1d/OOUI_JS_signature_icon_LTR.svg/44px-OOUI_JS_signature_icon_LTR.svg.png 2x" data-file-width="20" data-file-height="20" /></a> que encontrará en la barra de tareas cuando edite texto. En los artículos, sin embargo, no se espera que haya firmas y, de hecho, existe una política <a href="/wiki/Wikipedia:Acerca_de_firmar_art%C3%ADculos" title="Wikipedia:Acerca de firmar artículos">aprobada por la mayoría</a> de los colaboradores de Wikipedia en contra de las firmas en artículos.
</p><p>Adicionalmente, el uso de sangría puede mejorar considerablemente la distribución de una página de discusión, haciéndola más fácil de leer. Un estándar practicado en Wikipedia es poner sangría a tus comentarios un nivel más que el comentario de la persona a la que estás respondiendo. Puedes llevarlo a la práctica siguiendo las recomendaciones mencionadas en el apartado de <a href="#Sangría">sangría</a>.
</p>

<h3><span id="C.C3.B3mo_se_edita_una_redirecci.C3.B3n"></span><span class="mw-headline" id="Cómo_se_edita_una_redirección">Cómo se edita una redirección</span></h3>
<div class="noprint AP rellink"><span style="font-size:88%">Artículo principal:</span>&#32;<i><a href="/wiki/Ayuda:Redirecciones" class="mw-redirect" title="Ayuda:Redirecciones"> Ayuda:Redirecciones</a></i></div>
<p>Si deseas redirigir el tráfico de una página hacia otra (alguna con un mejor título, por ejemplo), puedes usar el comando "<code>#REDIRECCIÓN: [[nombre de página]]</code>" al comienzo de la descripción de la página o bien, presionar el botón <img alt="Redirección" src="//upload.wikimedia.org/wikipedia/commons/1/13/Vector_toolbar_redirect_button.png" decoding="async" width="28" height="28" data-file-width="28" data-file-height="28" /> para añadirlo automáticamente; este botón se encuentra bajo la sección «Avanzado» de la <a href="/wiki/Ayuda:Barra_de_herramientas" class="mw-redirect" title="Ayuda:Barra de herramientas">barra de herramientas</a>.
</p><p>Ve a <a href="/wiki/Patata" class="mw-redirect" title="Patata">patata</a>, por ejemplo. Para consultar el historial de la página antigua, pulsa sobre el enlace <b>"Redirigido desde <u>página antigua</u>"</b> en la zona superior de la página nueva.
</p>
<h2><span class="mw-headline" id="Consejos_y_recomendaciones">Consejos y recomendaciones</span></h2>
<p>Prohibido:
</p>
<ul><li>La <b>propaganda</b>: este es un trabajo desinteresado y gratuito hecho con esfuerzo y no lucrativo.</li>
<li>El <b>insulto</b>, por razones obvias. Somos muchos con historias, vivencias, cultura y lugares diferentes... y distintas ideas.</li>
<li>La <b>copia</b>. No se puede copiar textos ni imágenes si tienen derechos de autor a menos que estos mismos derechos autoricen tal copia, y su modificación y redistribución de manera compatible con la licencia <a href="/wiki/GFDL" class="mw-redirect" title="GFDL">GFDL</a>.</li>
<li><b>Firmar</b> los artículos. Las enciclopedias en general y Wikipedia en particular son obras colaborativas; por lo tanto, nadie puede otorgarse su autoría. Sí deben firmarse los comentarios que se escriban en el <a href="/wiki/Wikipedia:Caf%C3%A9" title="Wikipedia:Café">Café</a>, en las votaciones y en la discusión de artículos y de páginas de otros usuarios.</li>
<li><b>En imágenes</b>, incluir las notas de copyright, autorización, autoría y enlaces de los pies de foto (véase <a href="/wiki/Wikipedia:Votaciones/2004/Sobre_el_uso_de_los_pies_de_foto" title="Wikipedia:Votaciones/2004/Sobre el uso de los pies de foto">Wikipedia:Votaciones/2004/Sobre el uso de los pies de foto</a>).</li></ul>
<p>Evita, dentro de los límites que dicta la lógica, escribir:
</p>
<ul><li><b>«Actualmente»</b>, pues pasado un tiempo queda obsoleto. Es mejor poner la fecha. Ejemplo: sustituir "La moneda actualmente es el euro" por "desde 1999 la moneda es el euro".</li>
<li><b>«Ahora»</b>: tal vez mañana ya no lo sea.</li>
<li><b>Comparaciones</b>: "Las <b>mejores</b> playas del mediterráneo" (Casi seguro que esto no sea cierto o cuanto menos sostenible en el tiempo).</li>
<li><b>Exageraciones</b>: "Podemos ver <b>todas</b> las aves" (algunas faltarán).</li>
<li><b>No escribas en primera persona</b>: no deben utilizarse ninguna de las primeras personas, ni del plural ni del singular, tipo “yo”, “nosotros”, “mi”, “nuestros”, etc.</li></ul>
<p>Modifica sin miedo, pero con justificación:
</p>
<ul><li>Es mejor no poner nada que sea un <b>dato erróneo</b>.</li>
<li>Pon un <b>resumen</b> (al lado del botón "Guardar cambios"): les ahorrarás tiempo a los bibliotecarios, y la descripción breve de tu cambio quedará incluida en el historial de la página/del artículo.</li>
<li>No <b>borres</b> nada si no has comprobado antes que era erróneo.</li></ul>
<p>Si son datos variables en el tiempo: 
</p>
<ul><li>Coloca la <b>fecha</b> en que se tomaron:</li></ul>
<dl><dd><dl><dd>ej. La población es de 10&#160;600 habitantes <font size="-1">(2003)</font></dd></dl></dd></dl>
<ul><li>Coloca la <b>fuente</b> de la que fueron tomadas:</li></ul>
<dl><dd><dl><dd>ej. La población es de 10&#160;600 habitantes <font size="-1">(2003 <a href="/wiki/Instituto_Nacional_de_Estad%C3%ADstica_de_Espa%C3%B1a" class="mw-redirect" title="Instituto Nacional de Estadística de España">Instituto Nacional de Estadística</a>)</font></dd></dl></dd></dl>
<p>Es de muy mal gusto:
</p>
<ul><li><b>Corregir</b> algo escrito en el <a href="/wiki/Wikipedia:Caf%C3%A9" title="Wikipedia:Café">Café</a> (excepto casos extremos) sin el permiso del autor. No olvides que errar es de humanos... (la prisa y las ganas hacen cometer faltas ortográficas).</li>
<li><b>No firmar</b> cuando se hacen comentarios en el <a href="/wiki/Wikipedia:Caf%C3%A9" title="Wikipedia:Café">Café</a> o en la discusión de artículos y de páginas de otros usuarios: es como tirar la piedra y esconder la mano (no decir ni por qué ni quién).</li></ul>
<p>La virtud más apreciada es la <b>paciencia</b>. Encontrarás textos desviados, con malas ideas, provocadores, enjuiciadores, insultantes, vandálicos... a esto... paciencia. Y si quieres y puedes, corrígelo siguiendo las normas del <a href="/wiki/Wikipedia:Manual_de_estilo" title="Wikipedia:Manual de estilo">Manual de estilo</a> de la <a href="/wiki/Wikipedia" title="Wikipedia">Wikipedia</a>.
</p><p>En Wikipedia suelen realizarse <a href="/wiki/Wikipedia:Votaciones" title="Wikipedia:Votaciones">votaciones</a>, sobre todo para buscar el consenso. Para eso debes de saber cómo funcionan. Se considera que con 500 ediciones y seis meses de antigüedad puedes votar. (Véase <a href="/wiki/Wikipedia:Votaciones/2015/Requisitos_m%C3%ADnimos_para_derecho_a_voto" title="Wikipedia:Votaciones/2015/Requisitos mínimos para derecho a voto">Wikipedia:Votaciones/2015/Requisitos mínimos para derecho a voto</a>.)
</p>

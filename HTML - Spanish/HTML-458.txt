<h2><span class="mw-headline" id="Manual_de_Uso_de_Wikilibros"><b>Manual de Uso de Wikilibros</b></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Wikilibros:Manual&amp;action=edit&amp;section=T-1" title="Editar sección: ">editar</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><u>Objetivo del Manual de <i>Wikilibros</i>:</u>
Se ha creado este manual con el fin de reunificar las páginas de <a class="mw-selflink selflink">ayuda</a> de Wikilibros.
Aquí se puede encontrar, en un solo lugar, toda la información incluida en las diferentes secciones de ayuda.
Se espera que esta página sea de utilidad para las personas que quieran contribuir con el proyecto de <b>Wikilibros</b>. El índice de este manual está en: <br /><a class="mw-selflink selflink">Ayuda</a> y <a href="/wiki/Wikilibros:Manual" title="Wikilibros:Manual">Wikilibros:Manual</a>.
</p><p>Se puede añadir preguntas en <b><a href="/wiki/Wikilibros:Preguntas_frecuentes" class="mw-redirect" title="Wikilibros:Preguntas frecuentes">Preguntas frecuentes</a></b>.<br />
Si se desea obtener más ayuda, se puede contactar a usuarios más avanzados en <a href="/wiki/Wikilibros:Caf%C3%A9" title="Wikilibros:Café">el café</a>.
</p><p><u>Índice del Manual de <i>Wikilibros</i>:</u>

<h2 style="text-align: center; color: #006699;"><span class="mw-headline" id="Truco_del_mes">Truco del mes</span></h2>
<div style="border: 1px solid #e8cc88; background-color:#fff8ee; padding:1em; margin-top: 1em;"><div class="floatleft"><a href="/wiki/Archivo:Crystal_Clear_app_ktip.png" class="image"><img alt="Crystal Clear app ktip.png" src="//upload.wikimedia.org/wikipedia/commons/thumb/5/52/Crystal_Clear_app_ktip.png/50px-Crystal_Clear_app_ktip.png" decoding="async" width="50" height="50" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/5/52/Crystal_Clear_app_ktip.png/75px-Crystal_Clear_app_ktip.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/5/52/Crystal_Clear_app_ktip.png/100px-Crystal_Clear_app_ktip.png 2x" data-file-width="128" data-file-height="128" /></a></div>
<p><b>Crear una página que contenga todo el texto de las subpáginas de un libro</b>
</p><p>Normalmente los wikilibros largos tienen el contenido repartido en varias <a href="/wiki/Wikilibros:Pol%C3%ADtica_de_nombrado" title="Wikilibros:Política de nombrado">subpáginas</a>. Resulta útil tener una versión del libro con todo el contenido en una página para imprimirlo o bien para guardarlo y consultarlo <i>offline</i>.
</p><p>Para ello debes crear una nueva subpágina del libro que puedes llamar <b><i>Nombre del libro</i>/Texto completo</b>. Como contenido puedes poner:
</p>
<pre>{{Versión para imprimir|<i>Título del libro</i>}}
</pre>
<p>Y en vez de copiar y pegar el contenido de las subpáginas, que provocaría tener información duplicada, se puede hacer una referencia a su contenido usando las subpáginas como plantillas. Para mostrar el contenido de cada subpágina utiliza la siguiente sintaxis:
</p>
<pre>{{:<i>Nombre del libro</i>/<i>Subpágina</i>}}
</pre>
<p>Además, dentro de las subpáginas, puedes encerrar todo el texto que no quieras que aparezca en la página que contiene todo el libro (como plantillas de navegación o títulos de nivel superior) entre las etiquetas <tt>&lt;noinclude&gt;</tt> y <tt>&lt;/noinclude&gt;</tt>. De esta forma sólo aparecerán en las subpáginas pero no en la página que contiene todo el libro.
</p><p>Como los <a href="https://es.wikipedia.org/wiki/Ayuda:C%C3%B3mo_se_edita_una_p%C3%A1gina#Secciones" class="extiw" title="w:Ayuda:Cómo se edita una página">subtítulos</a> de nivel superior de las subpáginas suelen estar escritos entre signos "igual" dobles, puedes separar las subpáginas en la página que contiene todo el texto con subtítulos escritos entre signos "igual" simples, para que la tabla de contenidos respete una jerarquía lógica.
</p><p>Algunos ejemplos son <a href="/wiki/Programaci%C3%B3n_en_JavaScript/Texto_completo" title="Programación en JavaScript/Texto completo">Programación en JavaScript/Texto completo</a> y <a href="/wiki/Programaci%C3%B3n_en_Ada/Texto_completo" title="Programación en Ada/Texto completo">Programación en Ada/Texto completo</a>.
</p>

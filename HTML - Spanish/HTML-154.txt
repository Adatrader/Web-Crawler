<p>El <b>27 de junio</b> es el 178.º (centésimo septuagésimo octavo) <a href="/wiki/D%C3%ADa" title="Día">día</a> del <a href="/wiki/A%C3%B1o" title="Año">año</a> en el <a href="/wiki/Calendario_gregoriano" title="Calendario gregoriano">calendario gregoriano</a> y el 179.º en los <a href="/wiki/A%C3%B1os_bisiestos" class="mw-redirect" title="Años bisiestos">años bisiestos</a>. Quedan -8&#160;días para finalizar el año.
</p>

<h2><span class="mw-headline" id="Acontecimientos">Acontecimientos</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=27_de_junio&amp;action=edit&amp;section=1" title="Editar sección: Acontecimientos">editar</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul><li><a href="/wiki/1209" title="1209">1209</a>: se firma el <a href="/wiki/Tratado_de_Valladolid" title="Tratado de Valladolid">Tratado de Valladolid</a> entre los reyes <a href="/wiki/Alfonso_IX_de_Le%C3%B3n" title="Alfonso IX de León">Alfonso IX de León</a> y <a href="/wiki/Alfonso_VIII_de_Castilla" title="Alfonso VIII de Castilla">Alfonso VIII de Castilla</a> con el que se pone término a las disputas existentes entre ambos reinos.</li>
<li><a href="/wiki/1358" title="1358">1358</a>: nace la <a href="/wiki/Rep%C3%BAblica_de_Dubrovnik" class="mw-redirect" title="República de Dubrovnik">República de Dubrovnik</a>.</li>
<li><a href="/wiki/1410" title="1410">1410</a>: en <a href="/wiki/M%C3%A1laga" title="Málaga">Málaga</a> (España), las tropas <a href="/wiki/Nazar%C3%AD" title="Nazarí">nazaritas</a> incendian las torres de combate cristianas que mantenían el asedio a <a href="/wiki/Antequera" title="Antequera">Antequera</a>.</li>
<li><a href="/wiki/1506" title="1506">1506</a>: se firma la <a href="/wiki/Concordia_de_Villaf%C3%A1fila" title="Concordia de Villafáfila">Concordia de Villafáfila</a> en la localidad de Villafáfila donde <a href="/wiki/Fernando_el_Cat%C3%B3lico" class="mw-redirect" title="Fernando el Católico">Fernando el Católico</a> y <a href="/wiki/Felipe_I_de_Castilla" title="Felipe I de Castilla">Felipe el Hermoso</a>, se reconocía la incapacidad de <a href="/wiki/Juana_la_Loca" class="mw-redirect" title="Juana la Loca">Juana la Loca</a> para reinar debido a su enajenación mental.</li>
<li><a href="/wiki/1542" title="1542">1542</a>: <a href="/wiki/Juan_Rodr%C3%ADguez_Cabrillo" title="Juan Rodríguez Cabrillo">Juan Rodríguez Cabrillo</a> reclama la soberanía española de <a href="/wiki/California" title="California">California</a>.</li>
<li><a href="/wiki/1706" title="1706">1706</a>: en <a href="/wiki/Espa%C3%B1a" title="España">España</a> ante el sesgo que toma la guerra de sucesión se anuncia el traslado de la Corte de <a href="/wiki/Madrid" title="Madrid">Madrid</a> a <a href="/wiki/Burgos" title="Burgos">Burgos</a>.</li>
<li><a href="/wiki/1709" title="1709">1709</a>: <a href="/wiki/Pedro_I_de_Rusia" title="Pedro I de Rusia">Pedro el Grande</a> derrota a <a href="/wiki/Carlos_XII_de_Suecia" title="Carlos XII de Suecia">Carlos XII de Suecia</a> en la <a href="/wiki/Batalla_de_Poltava" title="Batalla de Poltava">Batalla de Poltava</a>.</li></ul>

<ul><li><a href="/wiki/1743" title="1743">1743</a>: <a href="/wiki/Guerra_de_Sucesi%C3%B3n_Austriaca" class="mw-redirect" title="Guerra de Sucesión Austriaca">Guerra de Sucesión Austriaca</a>: <a href="/wiki/Batalla_de_Dettingen" title="Batalla de Dettingen">Batalla de Dettingen</a>: en el campo de batalla de Bavária, <a href="/wiki/Jorge_II_de_Gran_Breta%C3%B1a" title="Jorge II de Gran Bretaña">Jorge II de Gran Bretaña</a> en persona lidera las tropas a la batalla. Es la última vez que un rey británico comanda sus tropas en el campo de batalla.</li>
<li><a href="/wiki/1759" title="1759">1759</a>: el general <a href="/wiki/James_Wolfe" title="James Wolfe">James Wolfe</a> inicia el asedio de <a href="/wiki/Quebec" title="Quebec">Quebec</a>.</li>
<li><a href="/wiki/1806" title="1806">1806</a>: en el marco de las <a href="/wiki/Invasiones_inglesas" class="mw-redirect" title="Invasiones inglesas">Invasiones inglesas</a> al Río de la Plata, la ciudad de <a href="/wiki/Buenos_Aires" title="Buenos Aires">Buenos Aires</a> es ocupada por tropas británicas.</li>
<li><a href="/wiki/1818" title="1818">1818</a>: en Caracas (Venezuela) se imprime el primer ejemplar del periódico <i><a href="/wiki/Correo_del_Orinoco" title="Correo del Orinoco">Correo del Orinoco</a></i>, fundado por <a href="/wiki/Sim%C3%B3n_Bol%C3%ADvar" title="Simón Bolívar">Simón Bolívar</a>.</li>
<li><a href="/wiki/1839" title="1839">1839</a>: en <a href="/wiki/Buenos_Aires" title="Buenos Aires">Buenos Aires</a> (Argentina) es asesinado el político <a href="/wiki/Manuel_Vicente_Maza" title="Manuel Vicente Maza">Manuel Vicente Maza</a> por conspirar contra el Gobierno popular de <a href="/wiki/Juan_Manuel_de_Rosas" title="Juan Manuel de Rosas">Juan Manuel de Rosas</a>.</li>
<li><a href="/wiki/1893" title="1893">1893</a>: crash de la <a href="/wiki/Bolsa_de_Nueva_York" title="Bolsa de Nueva York">Bolsa de Nueva York</a>.</li>

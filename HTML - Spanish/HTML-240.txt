<p>El <b>Mediterráneo</b> es un <a href="/wiki/Mar" title="Mar">mar</a> interior del océano <a href="/wiki/Oc%C3%A9ano_Atl%C3%A1ntico" title="Océano Atlántico">Atlántico</a>, con el que se conecta a través del <a href="/wiki/Estrecho_de_Gibraltar" title="Estrecho de Gibraltar">estrecho de Gibraltar</a>. Rodeado por <a href="/wiki/Europa" title="Europa">Europa</a>, <a href="/wiki/%C3%81frica" title="África">África</a> y <a href="/wiki/Asia" title="Asia">Asia</a>, fue testigo de la evolución de varias civilizaciones antiguas como los <a href="/wiki/Antiguo_Egipto" title="Antiguo Egipto">egipcios</a>, los <a href="/wiki/Fenicia" title="Fenicia">fenicios</a>, <a href="/wiki/Tierra_de_Israel" title="Tierra de Israel">hebreos</a>, <a href="/wiki/Antigua_Grecia" title="Antigua Grecia">griegos</a>, <a href="/wiki/Estado_p%C3%BAnico" title="Estado púnico">cartagineses</a> y <a href="/wiki/Antigua_Roma" title="Antigua Roma">romanos</a>. Con aproximadamente 2,5 millones de km² y <span style="white-space:nowrap">3860 km</span> de longitud, es el segundo mar interior más grande del mundo, después del <a href="/wiki/Mar_Caribe" title="Mar Caribe">Caribe</a>.<sup id="cite_ref-Durvan_1-0" class="reference separada"><a href="#cite_note-Durvan-1"><span class="corchete-llamada">[</span>1<span class="corchete-llamada">]</span></a></sup>&#8203; Se trata de un mar relativamente hondo, con una profundidad media de 1370 metros, siendo su punto más profundo la <a href="/wiki/Fosa_de_Calipso" title="Fosa de Calipso">fosa de Calipso</a>, al oeste de Grecia. Sus aguas, que bañan las tres grandes penínsulas del sur de Europa (<a href="/wiki/Pen%C3%ADnsula_ib%C3%A9rica" title="Península ibérica">ibérica</a>, <a href="/wiki/Pen%C3%ADnsula_It%C3%A1lica" class="mw-redirect" title="Península Itálica">itálica</a> y <a href="/wiki/Pen%C3%ADnsula_Balc%C3%A1nica" class="mw-redirect" title="Península Balcánica">balcánica</a>) y una de Asia (<a href="/wiki/Anatolia" title="Anatolia">Anatolia</a>), se comunican, además de con el Atlántico, con el <a href="/wiki/Mar_Negro" title="Mar Negro">mar Negro</a> por los estrechos del <a href="/wiki/B%C3%B3sforo" title="Bósforo">Bósforo</a> y de los <a href="/wiki/Dardanelos" title="Dardanelos">Dardanelos</a> y con el <a href="/wiki/Mar_Rojo" title="Mar Rojo">mar Rojo</a> por el <a href="/wiki/Canal_de_Suez" title="Canal de Suez">canal de Suez</a>.<sup id="cite_ref-2" class="reference separada"><a href="#cite_note-2"><span class="corchete-llamada">[</span>2<span class="corchete-llamada">]</span></a></sup>&#8203;
</p><p>Se trata de un mar cálido con un <a href="/wiki/Clima_mediterr%C3%A1neo" title="Clima mediterráneo">clima característico homónimo</a> y un <a href="/wiki/Dieta_mediterr%C3%A1nea" title="Dieta mediterránea">estilo culinario propio</a> en toda <a href="/wiki/Cuenca_del_Mediterr%C3%A1neo" title="Cuenca del Mediterráneo">su cuenca</a>. Estos factores atraen a millones de turistas de otros puntos del mundo. Por otro lado, es el mar con las tasas más elevadas de <a href="/wiki/Hidrocarburo" title="Hidrocarburo">hidrocarburos</a> y <a href="/wiki/Contaminaci%C3%B3n_marina" title="Contaminación marina">contaminación</a> del mundo.<sup id="cite_ref-3" class="reference separada"><a href="#cite_note-3"><span class="corchete-llamada">[</span>3<span class="corchete-llamada">]</span></a></sup>&#8203;
</p>

<h2><span class="mw-headline" id="Nombre">Nombre</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mar_Mediterr%C3%A1neo&amp;action=edit&amp;section=1" title="Editar sección: Nombre">editar</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>La <a href="/wiki/Etimolog%C3%ADa" title="Etimología">etimología</a> de este mar proviene del <a href="/wiki/Lat%C3%ADn" title="Latín">latín</a> «<i>Mar Medi Terraneum</i>», cuyo significado es «mar en el medio de las tierras». La mayoría de los habitantes de sus costas han usado una denominación derivada de la latina; en <a href="/wiki/Idioma_griego" title="Idioma griego">idioma griego</a> se llama «<i>Mesogeios Thalassa</i>» (Μεσόγειος Θάλασσα) con el mismo significado que el nombre latino, y en <a href="/wiki/Idioma_%C3%A1rabe" title="Idioma árabe">árabe</a> se llama <i>al-Baḥr al-Mutawāsiṭ</i> (البحر المتواسط «mar intermedio»).
</p><p>En <a href="/wiki/Idioma_turco" title="Idioma turco">turco</a>, se lo denomina «<i>Ak Deniz</i>«&#160;(«mar blanco») por oposición al <a href="/wiki/Mar_Negro" title="Mar Negro">mar Negro</a>, pues los turcos llaman «blanco» al Sur y «negro» al Norte. El nombre turco también se utiliza a veces en árabe: <i>al-Baḥr al-′Ābyaḍ</i> (البحر الأبيض ídem).<sup id="cite_ref-4" class="reference separada"><a href="#cite_note-4"><span class="corchete-llamada">[</span>4<span class="corchete-llamada">]</span></a></sup>&#8203;
</p><p>Para los <a href="/wiki/Antiguo_Egipto" title="Antiguo Egipto">antiguos egipcios</a>, era «el Gran Verde». Los romanos lo solían llamar «<i>Mare Nostrum</i>» («mar nuestro»). El uso más antiguo del adjetivo <i>mediterraneum</i> para referirse a este mar data de <a href="/wiki/Solino" class="mw-redirect" title="Solino">Solino</a>, en el siglo&#160;<span style="font-variant:small-caps;text-transform:lowercase">III</span> d.C., y el primero que usó <i>Mediterraneum</i> como nombre propio fue <a href="/wiki/Isidoro_de_Sevilla" title="Isidoro de Sevilla">Isidoro de Sevilla</a>, en el siglo&#160;<span style="font-variant:small-caps;text-transform:lowercase">VI</span>.<sup id="cite_ref-5" class="reference separada"><a href="#cite_note-5"><span class="corchete-llamada">[</span>5<span class="corchete-llamada">]</span></a></sup>&#8203; Los árabes, durante la Edad Media, usaban mayoritariamente el nombre de <i>al-Baḥr al-Rum</i> («mar de los europeos no musulmanes»), siguiendo la costumbre de nombrar a mares y océanos con las poblaciones que ocupaban las orillas opuestas. Otros nombres árabes medievales fueron <i>al-Baḥr al-Sham</i> («mar de Siria», preferido por los habitantes del Mediterráneo occidental), <i>al-Baḥr al-Maghrib</i> («mar de poniente» o «del Magreb», popular entre los árabes del Mediterráneo oriental) y, muy minoritario, el ya citado <i>al-Baḥr al-Mutawāsiṭ</i>.<sup id="cite_ref-6" class="reference separada"><a href="#cite_note-6"><span class="corchete-llamada">[</span>6<span class="corchete-llamada">]</span></a></sup>&#8203;
</p>
<h2><span id="Situaci.C3.B3n"></span><span class="mw-headline" id="Situación">Situación</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mar_Mediterr%C3%A1neo&amp;action=edit&amp;section=2" title="Editar sección: Situación">editar</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="thumb tleft"><div class="thumbinner" style="width:222px;"><a href="/wiki/Archivo:Mediterranean_Sea_Bathymetry_map.svg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/f/f9/Mediterranean_Sea_Bathymetry_map.svg/220px-Mediterranean_Sea_Bathymetry_map.svg.png" decoding="async" width="220" height="103" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/f/f9/Mediterranean_Sea_Bathymetry_map.svg/330px-Mediterranean_Sea_Bathymetry_map.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/f/f9/Mediterranean_Sea_Bathymetry_map.svg/440px-Mediterranean_Sea_Bathymetry_map.svg.png 2x" data-file-width="4252" data-file-height="2000" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Archivo:Mediterranean_Sea_Bathymetry_map.svg" class="internal" title="Aumentar"></a></div>Mapa <a href="/wiki/Is%C3%B3bata" title="Isóbata">isobático</a> del Mediterráneo</div></div></div>
<p>El mar Mediterráneo se encuentra localizado en las zonas templadas y subtropicales. Está rodeado por tierras continentales de amplia extensión, dotándolo de una climatología propia caracterizada por veranos cálidos y secos, inviernos moderados y un periodo que concentra buena parte de las lluvias durante la primavera.<sup id="cite_ref-saliot_7-0" class="reference separada"><a href="#cite_note-saliot-7"><span class="corchete-llamada">[</span>7<span class="corchete-llamada">]</span></a></sup>&#8203; La urbanización costera, típica en las zonas mediterráneas, acomoda a un buen número de habitantes permanentes, amén de cierto número de población temporal, especialmente estival. Las zonas más septentrionales, muy industrializadas, contrastan con las más meridionales, esencialmente agrícolas.<sup id="cite_ref-8" class="reference separada"><a href="#cite_note-8"><span class="corchete-llamada">[</span>8<span class="corchete-llamada">]</span></a></sup>&#8203;
</p><p>El mar Mediterráneo posee una extensión de <span style="white-space:nowrap">3,5 x 10<sup>6</sup> km²</span>, lo cual representa aproximadamente el 1% de la superficie oceánica mundial, con una profundidad media de 1,5 km. Su zona más profunda, en <a href="/wiki/Cabo_T%C3%A9naro" title="Cabo Ténaro">Matapan</a>, alcanza <span style="white-space:nowrap">5121 m</span> bajo el nivel del mar. Su longitud costera es de <span style="white-space:nowrap">46 000 km</span>, de los cuales el 40&#160;% pertenecen a islas.<sup id="cite_ref-saliot_7-1" class="reference separada"><a href="#cite_note-saliot-7"><span class="corchete-llamada">[</span>7<span class="corchete-llamada">]</span></a></sup>&#8203;
</p><p>Comunica con el <a href="/wiki/Oc%C3%A9ano_Atl%C3%A1ntico" title="Océano Atlántico">océano Atlántico</a> mediante el <a href="/wiki/Estrecho_de_Gibraltar" title="Estrecho de Gibraltar">estrecho de Gibraltar</a>, un paso de <span style="white-space:nowrap">12,8 km</span> de longitud y una profundidad de más de <span style="white-space:nowrap">900 m</span>; por el noroeste, contacta con el <a href="/wiki/Mar_Negro" title="Mar Negro">mar Negro</a> mediante el <a href="/wiki/Estrecho_de_Dardanelos" class="mw-redirect" title="Estrecho de Dardanelos">estrecho de Dardanelos</a>, el <a href="/wiki/Mar_de_M%C3%A1rmara" title="Mar de Mármara">mar de Mármara</a> y el <a href="/wiki/Estrecho_del_B%C3%B3sforo" class="mw-redirect" title="Estrecho del Bósforo">estrecho del Bósforo</a>. En el sureste, contacta con el <a href="/wiki/Mar_Rojo" title="Mar Rojo">mar Rojo</a> debido a la acción humana, que excavó el <a href="/wiki/Canal_de_Suez" title="Canal de Suez">canal de Suez</a>, canal que sirve de entrada a especies tropicales del mar Rojo y del <a href="/wiki/Oc%C3%A9ano_%C3%8Dndico" title="Océano Índico">océano Índico</a>.<sup id="cite_ref-saliot_7-2" class="reference separada"><a href="#cite_note-saliot-7"><span class="corchete-llamada">[</span>7<span class="corchete-llamada">]</span></a></sup>&#8203;
</p>

<li>El <a href="/wiki/Mar_Tirreno" title="Mar Tirreno">mar Tirreno</a> entre la costa este de Cerdeña, la península itálica y la costa norte de Sicilia.</li>
<li>El <a href="/wiki/Mar_Adri%C3%A1tico" title="Mar Adriático">mar Adriático</a> entre la península itálica y las costas de Eslovenia, Croacia, Bosnia y Montenegro.</li>
<li>El <a href="/wiki/Mar_J%C3%B3nico" title="Mar Jónico">mar Jónico</a> entre la península itálica, Grecia y Albania.</li>
<li>El <a href="/wiki/Mar_Egeo" title="Mar Egeo">mar Egeo</a> entre Grecia y Turquía.
<ul><li>El <a href="/wiki/Mar_de_Creta" title="Mar de Creta">mar de Creta</a> entre la isla de <a href="/wiki/Creta" title="Creta">Creta</a> y las islas <a href="/wiki/C%C3%ADcladas" title="Cícladas">Cícladas</a>.</li></ul></li>

<h2><span class="mw-headline" id="Historia">Historia</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mar_Mediterr%C3%A1neo&amp;action=edit&amp;section=8" title="Editar sección: Historia">editar</a><span class="mw-editsection-bracket">]</span></span></h2>
<div class="noprint AP rellink"><span style="font-size:88%">Artículo principal:</span>&#32;<i><a href="/wiki/Cuenca_del_Mediterr%C3%A1neo" title="Cuenca del Mediterráneo"> Cuenca del Mediterráneo</a></i></div><div class="thumb tright"><div class="thumbinner" style="width:222px;"><a href="/wiki/Archivo:Corbita_BM_GR1850.3-4.32.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/7/70/Corbita_BM_GR1850.3-4.32.jpg/220px-Corbita_BM_GR1850.3-4.32.jpg" decoding="async" width="220" height="126" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/7/70/Corbita_BM_GR1850.3-4.32.jpg/330px-Corbita_BM_GR1850.3-4.32.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/7/70/Corbita_BM_GR1850.3-4.32.jpg/440px-Corbita_BM_GR1850.3-4.32.jpg 2x" data-file-width="2400" data-file-height="1375" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Archivo:Corbita_BM_GR1850.3-4.32.jpg" class="internal" title="Aumentar"></a></div>Talla de un pequeño velero encontrada en la antigua <a href="/wiki/Cartago" title="Cartago">Cartago</a>, hacia el 200&#160;a.&#160;C. Época de disputas por la hegemonía en el Mediterráneo de cartagineses y romanos.</div></div></div><p>El mar Mediterráneo ha sido un mar clave para la <a href="/wiki/Historia" title="Historia">Historia</a>. <a href="/wiki/Fenicia" title="Fenicia">Fenicios</a> y <a href="/wiki/Antigua_Roma" title="Antigua Roma">romanos</a> navegaron por él junto a los <a href="/wiki/Grecia_Antigua" class="mw-redirect" title="Grecia Antigua">griegos</a>, quienes sobrepasaron los límites del mismo a través del <a href="/wiki/Estrecho_de_Gibraltar" title="Estrecho de Gibraltar">estrecho de Gibraltar</a> hace unos tres mil años aproximadamente. Estos navegantes solo conocían las corrientes de los ríos y supusieron que la extensión de agua al otro lado de Gibraltar era un enorme río. Por lo tanto, la palabra que significa <i>río</i> en el griego clásico era <i>Okeano</i>, y de allí proviene la palabra <i>océano</i>.<sup id="cite_ref-thalassa_23-0" class="reference separada"><a href="#cite_note-thalassa-23"><span class="corchete-llamada">[</span>23<span class="corchete-llamada">]</span></a></sup>&#8203;
</p><p>En la <a href="/wiki/Roma_antigua" class="mw-redirect" title="Roma antigua">Roma antigua</a> se llamó «Mare Nostrum» (<i>Nuestro mar</i>) debido a que todas sus orillas fueron ocupadas por ella,<sup id="cite_ref-Hispánica_24-0" class="reference separada"><a href="#cite_note-Hispánica-24"><span class="corchete-llamada">[</span>24<span class="corchete-llamada">]</span></a></sup>&#8203; y en el siglo&#160;<span style="font-variant:small-caps;text-transform:lowercase">XX</span> <a href="/wiki/Mussolini" class="mw-redirect" title="Mussolini">Mussolini</a> —queriendo recrear el Imperio romano— lo llamó <a href="/wiki/Mare_Nostrum_Italiano" title="Mare Nostrum Italiano">Mare Nostrum Italiano</a> durante la Segunda Guerra Mundial. El mar originó una serie de leyendas y mitos en torno a él, como la de <a href="/wiki/Jas%C3%B3n" title="Jasón">Jasón</a> y los <a href="/wiki/Argonautas" title="Argonautas">argonautas</a>.<sup id="cite_ref-thalassa_23-1" class="reference separada"><a href="#cite_note-thalassa-23"><span class="corchete-llamada">[</span>23<span class="corchete-llamada">]</span></a></sup>&#8203;
</p>
<h2><span class="mw-headline" id="Naturaleza">Naturaleza</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mar_Mediterr%C3%A1neo&amp;action=edit&amp;section=9" title="Editar sección: Naturaleza">editar</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>La región mediterránea está caracterizada por un alto grado de <a href="/wiki/Endemismo" title="Endemismo">endemismo</a> que ha ocasionado que se la considere como uno de los lugares de mayor concentración de <a href="/wiki/Biodiversidad" title="Biodiversidad">biodiversidad</a> en el ámbito mundial.<sup id="cite_ref-iucn-es_25-0" class="reference separada"><a href="#cite_note-iucn-es-25"><span class="corchete-llamada">[</span>25<span class="corchete-llamada">]</span></a></sup>&#8203; Dicha peculiaridad se explica de diversas formas: bajo impacto de la última <a href="/wiki/Glaciaci%C3%B3n" title="Glaciación">glaciación</a> en la zona, que produjo que actuara como reserva; la presencia de macizos montañosos importantes (por ejemplo, el <a href="/wiki/Atlas_(cordillera)" title="Atlas (cordillera)">Atlas</a>, la <a href="/wiki/Montes_Tauro" title="Montes Tauro">cordillera del Taurus</a> al sur, las zonas de <a href="/wiki/Sierra_de_G%C3%BAdar" title="Sierra de Gúdar">Gúdar</a>, <a href="/wiki/Javalambre" class="mw-redirect" title="Javalambre">Javalambre</a>, Levante…), que dota de diversidad estructural al <a href="/wiki/Biotopo" title="Biotopo">biotopo</a>; y también a la larga historia de los diversos usos de la tierra por parte de la población local que ha creado y mantenido una amplia gama de <a href="/wiki/H%C3%A1bitat" title="Hábitat">hábitats</a>.<sup id="cite_ref-iucn-es_25-1" class="reference separada"><a href="#cite_note-iucn-es-25"><span class="corchete-llamada">[</span>25<span class="corchete-llamada">]</span></a></sup>&#8203;
</p>
<h3><span id="Geolog.C3.ADa"></span><span class="mw-headline" id="Geología">Geología</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mar_Mediterr%C3%A1neo&amp;action=edit&amp;section=10" title="Editar sección: Geología">editar</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright"><div class="thumbinner" style="width:222px;"><a href="/wiki/Archivo:Mediterranean_Rupelian.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/1/12/Mediterranean_Rupelian.jpg/220px-Mediterranean_Rupelian.jpg" decoding="async" width="220" height="158" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/1/12/Mediterranean_Rupelian.jpg/330px-Mediterranean_Rupelian.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/1/12/Mediterranean_Rupelian.jpg/440px-Mediterranean_Rupelian.jpg 2x" data-file-width="1038" data-file-height="747" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Archivo:Mediterranean_Rupelian.jpg" class="internal" title="Aumentar"></a></div>Mapa del primitivo mar <a href="/wiki/Paratetis" title="Paratetis">Paratetis</a>, sucesor del <a href="/wiki/Oc%C3%A9ano_Tetis" title="Océano Tetis">océano Tetis</a>, y que formó el mar Mediterráneo a medida que se cerró. El mapa corresponde al <a href="/wiki/Oligoceno" title="Oligoceno">Oligoceno</a>.</div></div></div>
<p>El mar Mediterráneo se formó a partir del <a href="/wiki/Oc%C3%A9ano_Tetis" title="Océano Tetis">océano Tetis</a>, de mucha mayor extensión que el Mediterráneo actual, y fue disminuyendo a medida que la <a href="/wiki/Placa_Africana" class="mw-redirect" title="Placa Africana">placa Africana</a> se aproximaba a la <a href="/wiki/Placa_Euroasi%C3%A1tica" class="mw-redirect" title="Placa Euroasiática">placa Euroasiática</a>.<sup id="cite_ref-26" class="reference separada"><a href="#cite_note-26"><span class="corchete-llamada">[</span>26<span class="corchete-llamada">]</span></a></sup>&#8203;
</p><p>
Hace unos 60 millones de años, a comienzos del <a href="/wiki/Era_Terciaria" class="mw-redirect" title="Era Terciaria">Terciario</a>, el Océano Tetis, antecesor del Mediterráneo, inundaba gran parte del norte de <a href="/wiki/%C3%81frica" title="África">África</a> y <a href="/wiki/Europa" title="Europa">Europa</a>, que en aquel entonces no era más que un archipiélago de islas.<sup id="cite_ref-SAL_27-0" class="reference separada"><a href="#cite_note-SAL-27"><span class="corchete-llamada">[</span>27<span class="corchete-llamada">]</span></a></sup>&#8203; Durante el <a href="/wiki/Era_Terciaria" class="mw-redirect" title="Era Terciaria">Terciario</a>, el océano Tetis se fue estrechando por el este hasta que se formó una gran cuenca marina casi separada del océano abierto. Esta cuenca abarcaba en una misma masa de agua al Mediterráneo, al <a href="/wiki/Mar_Negro" title="Mar Negro">mar Negro</a> y al <a href="/wiki/Mar_Caspio" title="Mar Caspio">mar Caspio</a>. Pero por occidente el <i>Mare Nostrum</i> seguía conectado a mar abierto a través de superficies hoy en día emergidas del sur de la <a href="/wiki/Pen%C3%ADnsula_ib%C3%A9rica" title="Península ibérica">península ibérica</a> y el norte de <a href="/wiki/Marruecos" title="Marruecos">Marruecos</a>.<sup id="cite_ref-SAL_27-1" class="reference separada"><a href="#cite_note-SAL-27"><span class="corchete-llamada">[</span>27<span class="corchete-llamada">]</span></a></sup>&#8203;</p><div class="thumb tleft"><div class="thumbinner" style="width:222px;"><div id="mwe_player_0" class="PopUpMediaTransform" style="width:220px;" videopayload="&lt;div class=&quot;mediaContainer&quot; Este fenómeno generó enormes depósitos de sal en el fondo del Mediterráneo y disminuyó la salinidad del océano global<sup id="cite_ref-28" class="reference separada"><a href="#cite_note-28"><span class="corchete-llamada">[</span>28<span class="corchete-llamada">]</span></a></sup>&#8203;<sup id="cite_ref-SAL_27-2" class="reference separada"><a href="#cite_note-SAL-27"><span class="corchete-llamada">[</span>27<span class="corchete-llamada">]</span></a></sup>&#8203; También hoy en día, si el <a href="/wiki/Estrecho_de_Gibraltar" title="Estrecho de Gibraltar">estrecho de Gibraltar</a> se cerrara, el Mediterráneo se secaría puesto que es un mar en el que se evapora más agua (debido al clima cálido y templado) de la que entra por los ríos. Este déficit hídrico provoca actualmente una salinidad más elevada que en el Atlántico, océano del cual recibe el agua que pierde debido a la evaporación. El fenómeno inspiró el proyecto <a href="/wiki/Atlantropa" title="Atlantropa">Atlantropa</a>, que preveía la construcción de una presa en el <a href="/wiki/Estrecho_de_Gibraltar" title="Estrecho de Gibraltar">estrecho de Gibraltar</a> que permitiera producir energía y desecar parcialmente el mar. En un futuro (dentro de unos 5 millones de años) el <a href="/wiki/Estrecho_de_Gibraltar" title="Estrecho de Gibraltar">estrecho de Gibraltar</a> se cerrará nuevamente y el Mediterráneo quedará reducido a unos pocos <a href="/wiki/Lago" title="Lago">lagos</a> hipersalinos. Se sabe por los depósitos salinos del fondo del mar que esto ya sucedió hace unos pocos millones de años, concretamente 5,33. Entonces la diferencia de nivel entre el océano Atlántico y el mar Mediterráneo provocó la ruptura del dique de <a href="/wiki/Roca" title="Roca">roca</a> que se había formado en el Estrecho. Lo que había tardado miles de años en secarse se llenó en unos pocos años; se ha especulado con cifras de hasta dos años, ya que se ha podido especificar en un llenado de 10 metros diarios.<sup id="cite_ref-29" class="reference separada"><a href="#cite_note-29"><span class="corchete-llamada">[</span>29<span class="corchete-llamada">]</span></a></sup>&#8203;
</p><h3><span id="Oceanograf.C3.ADa"></span><span class="mw-headline" id="Oceanografía">Oceanografía</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mar_Mediterr%C3%A1neo&amp;action=edit&amp;section=11" title="Editar sección: Oceanografía">editar</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright"><div class="thumbinner" style="width:222px;"><a href="/wiki/Archivo:MEDCURR.GIF" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/ef/MEDCURR.GIF/220px-MEDCURR.GIF" decoding="async" width="220" height="111" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/ef/MEDCURR.GIF/330px-MEDCURR.GIF 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/ef/MEDCURR.GIF/440px-MEDCURR.GIF 2x" data-file-width="497" data-file-height="250" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Archivo:MEDCURR.GIF" class="internal" title="Aumentar"></a></div>Corrientes predominantes de junio</div></div></div>
<p>Las propiedades del mar Mediterráneo son consecuencia de su situación de mar casi interior. Así, las <a href="/wiki/Marea" title="Marea">mareas</a> son leves como resultado de la estrecha conexión con el <a href="/wiki/Oc%C3%A9ano_Atl%C3%A1ntico" title="Océano Atlántico">océano Atlántico</a>. El mediterráneo se caracteriza por su tono azulado, y es inmediatamente reconocido por este color particular, especialmente en las áreas cercanas a las islas griegas y croatas.
</p><p>La evaporación excede en grandes cantidades a las precipitaciones y las afluencias de ríos hacia el Mediterráneo, un factor que es central para la circulación del agua en la cuenca.<sup id="cite_ref-30" class="reference separada"><a href="#cite_note-30"><span class="corchete-llamada">[</span>30<span class="corchete-llamada">]</span></a></sup>&#8203; La evaporación es especialmente alta en la mitad este de la zona, causando que el nivel del agua descienda y la <a href="/wiki/Salinidad" title="Salinidad">salinidad</a> se incremente hacia el este.<sup id="cite_ref-31" class="reference separada"><a href="#cite_note-31"><span class="corchete-llamada">[</span>31<span class="corchete-llamada">]</span></a></sup>&#8203; Este gradiente de presión mantiene una baja salinidad proveniente desde el Atlántico hasta la cuenca, el cual se entibia y se vuelve más salado a medida que viaja hacia el este y baja en la región de <a href="/wiki/Levante_mediterr%C3%A1neo" title="Levante mediterráneo">Levante</a>, circulando luego hacia el oeste y dirigiéndose hasta el <a href="/wiki/Estrecho_de_Gibraltar" title="Estrecho de Gibraltar">estrecho de Gibraltar</a>.<sup id="cite_ref-32" class="reference separada"><a href="#cite_note-32"><span class="corchete-llamada">[</span>32<span class="corchete-llamada">]</span></a></sup>&#8203; De esta manera, la corriente marina fluye hacia el este bañando la superficie del estrecho y luego hacia el oeste por la parte baja. Una vez en mar abierto, esta "agua intermedia" puede persistir por miles de kilómetros alejada de su fuente principal.<sup id="cite_ref-33" class="reference separada"><a href="#cite_note-33"><span class="corchete-llamada">[</span>33<span class="corchete-llamada">]</span></a></sup>&#8203;
</p>
<h3><span class="mw-headline" id="Flora">Flora</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mar_Mediterr%C3%A1neo&amp;action=edit&amp;section=12" title="Editar sección: Flora">editar</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tleft"><div class="thumbinner" style="width:222px;"><a href="/wiki/Archivo:Seekiefern.JPG" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/b/b6/Seekiefern.JPG/220px-Seekiefern.JPG" decoding="async" width="220" height="165" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/b/b6/Seekiefern.JPG/330px-Seekiefern.JPG 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/b/b6/Seekiefern.JPG/440px-Seekiefern.JPG 2x" data-file-width="2272" data-file-height="1704" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Archivo:Seekiefern.JPG" class="internal" title="Aumentar"></a></div><i><a href="/wiki/Pinus_pinaster" title="Pinus pinaster">Pinus pinaster</a></i>, en la isla de <a href="/wiki/Tasos" title="Tasos">Tasos</a>, Grecia. Tiene presencia en todo el Mediterráneo, abundando en su parte occidental.</div></div></div><p>
Desde un punto de vista <a href="/wiki/Biogeograf%C3%ADa" title="Biogeografía">biogeográfico</a>, los territorios con clima mediterráneo ocupan no solo la cuenca del Mediterráneo, la Región Mediterránea propiamente dicha, sino que incluye también a las costas de <a href="/wiki/Libia" title="Libia">Libia</a> y <a href="/wiki/Egipto" title="Egipto">Egipto</a>, que pertenecen a la región saharo-arábiga. El clima mediterráneo se adentra hacia <a href="/wiki/Afganist%C3%A1n" title="Afganistán">Afganistán</a>, <a href="/wiki/C%C3%A1ucaso" title="Cáucaso">Cáucaso</a> y centro de Asia (donde <a href="/wiki/Armenia" title="Armenia">Armenia</a> y territorios de <a href="/wiki/Taskent" title="Taskent">Taskent</a> lo poseen, Región Irano-Turaniana).<sup id="cite_ref-izco_34-0" class="reference separada"><a href="#cite_note-izco-34"><span class="corchete-llamada">[</span>34<span class="corchete-llamada">]</span></a></sup>&#8203;</p><div class="thumb tright"><div class="thumbinner" style="width:222px;"><a href="/wiki/Archivo:Olive_niche.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/ab/Olive_niche.jpg/220px-Olive_niche.jpg" decoding="async" width="220" height="119" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/ab/Olive_niche.jpg/330px-Olive_niche.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/ab/Olive_niche.jpg/440px-Olive_niche.jpg 2x" data-file-width="744" data-file-height="403" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Archivo:Olive_niche.jpg" class="internal" title="Aumentar"></a></div>Distribución potencial del olivo en la Cuenca del Mediterráneo. El olivo se considera uno de los mejores indicadores biológicos de la región Mediterránea (Oteros, 2014).<sup id="cite_ref-Oteros_(2014)_35-0" class="reference separada"><a href="#cite_note-Oteros_(2014)-35"><span class="corchete-llamada">[</span>35<span class="corchete-llamada">]</span></a></sup>&#8203;</div></div></div>
<p>La Región Mediterránea posee característicamente un dosel arbóreo boscoso esclerófilo siempre verde, con especies del género <i><a href="/wiki/Quercus" title="Quercus">Quercus</a></i> del subgénero <i><a href="/w/index.php?title=Sclerophyllodrys&amp;action=edit&amp;redlink=1" class="new" title="Sclerophyllodrys (aún no redactado)">sclerophyllodrys</a></i>, especialmente <i><a href="/wiki/Quercus_ilex" title="Quercus ilex">Q. ilex</a></i>, la encina, aunque también posee mucha representación la carrasca, <i><a href="/wiki/Quercus_rotundifolia" class="mw-redirect" title="Quercus rotundifolia">Q. rotundifolia</a></i>. En otros casos aparecen el alcornoque, <i><a href="/wiki/Quercus_suber" title="Quercus suber">Q. suber</a></i>, dependiente de mayor homogeneidad en el ŕegimen pluvial, y, en el mediterráneo oriental, los vicariantes <i>Q. trojana</i> y <i>Q. macrolepis</i>. En cuanto a vegetación arbustiva, abunda la coscoja (<i><a href="/wiki/Quercus_coccifera" title="Quercus coccifera">Q. coccifera</a></i>). Dentro de este contexto ecológico de bosque de carrasca y encina, aparece una rica flora arbustiva de madroños (<i><a href="/wiki/Arbutus_unedo" title="Arbutus unedo">arbutus unedo</a></i>), viburnum tinus, <i><a href="/wiki/Laurus_nobilis" title="Laurus nobilis">laurus nobilis</a></i>, <i><a href="/wiki/Pistacia_lentiscus" title="Pistacia lentiscus">pistacia lentiscus</a></i>, rhamnus alaternus, etc. Como lianas, <i><a href="/wiki/Rubia_peregrina" title="Rubia peregrina">rubia peregrina</a></i>, <i><a href="/wiki/Lonicera" title="Lonicera">lonicera</a></i> sp. pl, <i><a href="/wiki/Smilax_aspera" title="Smilax aspera">smilax aspera</a></i> y otras, dependiendo de la humedad del ecosistema. En las áreas del sudeste peninsular aparecen especies más cercanas a la vegetación norteafricana, como <i><a href="/w/index.php?title=Ziziphyus_lotus&amp;action=edit&amp;redlink=1" class="new" title="Ziziphyus lotus (aún no redactado)">ziziphyus lotus</a></i>, <i><a href="/wiki/Periploca_angustifolia" title="Periploca angustifolia">periploca angustifolia</a></i> o <i><a href="/w/index.php?title=Maitenus_senegalensis&amp;action=edit&amp;redlink=1" class="new" title="Maitenus senegalensis (aún no redactado)">maitenus senegalensis</a></i>. Cuando el clima es más mesofítico y la aridez del verano es menor, surgen estructuras boscosas con predomino de robledales, formados por árboles marcescentes del género <i>quercus</i> como <i><a href="/wiki/Quercus_pyrenaica" title="Quercus pyrenaica">Q. pyrenaica</a></i> o <i><a href="/wiki/Quercus_faginea" title="Quercus faginea">Q. faginea</a></i>.<sup id="cite_ref-izco_34-1" class="reference separada"><a href="#cite_note-izco-34"><span class="corchete-llamada">[</span>34<span class="corchete-llamada">]</span></a></sup>&#8203;
</p><p>Los pinos mediterráneos, generalmente asociados a las especies <i><a href="/wiki/Pinus_halepensis" title="Pinus halepensis">pinus halepensis</a></i> y <i><a href="/wiki/Pinus_brutia" title="Pinus brutia">P. brutia</a></i>, también comprenden a <i><a href="/wiki/Pinus_pinaster" title="Pinus pinaster">P. pinaster</a></i>, en sustratos pobres en bases, y <i><a href="/wiki/Pinus_pinea" title="Pinus pinea">P. pinea</a></i>, en arenosos. En el piso supramediterráneo aparece <i><a href="/wiki/Pinus_nigra" title="Pinus nigra">P. nigra</a></i>, sobre sustratos ricos en bases, y en el oromediterráneo, <i><a href="/wiki/Pinus_sylvestris" title="Pinus sylvestris">P. sylvestris</a></i>. El género <i><a href="/wiki/Juniperus" title="Juniperus">juniperus</a></i>, de enebros y <a href="/wiki/Sabinares" class="mw-redirect" title="Sabinares">sabinares</a>, posee también una buena representación, con <i><a href="/wiki/Juniperus_oxycedrus" title="Juniperus oxycedrus">J. oxycedrus</a></i> como especie más común, si bien también son frecuentes <i><a href="/wiki/Juniperus_macrocarpa" title="Juniperus macrocarpa">J. macrocarpa</a></i>, <i><a href="/wiki/Juniperus_phoenicea" title="Juniperus phoenicea">J. phoenicea</a></i> y otros.
</p>
<h3><span class="mw-headline" id="Fauna">Fauna</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mar_Mediterr%C3%A1neo&amp;action=edit&amp;section=13" title="Editar sección: Fauna">editar</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright"><div class="thumbinner" style="width:222px;"><a href="/wiki/Archivo:Phoque_Moine_Monachus.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/8/81/Phoque_Moine_Monachus.jpg/220px-Phoque_Moine_Monachus.jpg" decoding="async" width="220" height="147" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/8/81/Phoque_Moine_Monachus.jpg/330px-Phoque_Moine_Monachus.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/8/81/Phoque_Moine_Monachus.jpg/440px-Phoque_Moine_Monachus.jpg 2x" data-file-width="600" data-file-height="400" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Archivo:Phoque_Moine_Monachus.jpg" class="internal" title="Aumentar"></a></div><a href="/wiki/Foca_monje_mediterr%C3%A1nea" class="mw-redirect" title="Foca monje mediterránea">Foca monje mediterránea</a>, actualmente en estado crítico de extinción.</div></div></div>
<p>La diversidad faunística del entorno mediterráneo es enorme, comparable a la de las plantas. Analizando la riqueza de especies para algunos taxones, obtenemos que: de las 62 especies de <a href="/wiki/Amphibia" title="Amphibia">anfibios</a> que hay en el Mediterráneo, 35 son endémicas, al igual que las 111 de las 179 especies de reptiles; de las 184 especies de <a href="/wiki/Mam%C3%ADfero" class="mw-redirect" title="Mamífero">mamíferos</a> registradas, el 25&#160;% son endémicas y 52 especies están amenazadas (sin contar los mamíferos marinos); y que el 28&#160;% de las especies marinas halladas en el Mediterráneo son endémicas.<sup id="cite_ref-iucn-es_25-2" class="reference separada"><a href="#cite_note-iucn-es-25"><span class="corchete-llamada">[</span>25<span class="corchete-llamada">]</span></a></sup>&#8203; En cuanto a las <a href="/wiki/Aves" title="Aves">aves</a>, el Mediterráneo es un mar de especial diversidad puesto que se halla en la ruta migratoria de multitud de especies. Se estima que unos 2.000 millones de aves migratorias de 150 especies distintas se detienen en sus <a href="/wiki/Humedal" title="Humedal">humedales</a> durante su trayecto o se establecen allí en algún periodo corto del año.<sup id="cite_ref-iucn-es_25-3" class="reference separada"><a href="#cite_note-iucn-es-25"><span class="corchete-llamada">[</span>25<span class="corchete-llamada">]</span></a></sup>&#8203;
</p>
<h3><span class="mw-headline" id="Amenazas">Amenazas</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mar_Mediterr%C3%A1neo&amp;action=edit&amp;section=14" title="Editar sección: Amenazas">editar</a><span class="mw-editsection-bracket">]</span></span></h3>
<div class="thumb tright"><div class="thumbinner" style="width:222px;"><a href="/wiki/Archivo:Mediterranean-major-cities.png" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/ab/Mediterranean-major-cities.png/220px-Mediterranean-major-cities.png" decoding="async" width="220" height="111" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/ab/Mediterranean-major-cities.png/330px-Mediterranean-major-cities.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/ab/Mediterranean-major-cities.png/440px-Mediterranean-major-cities.png 2x" data-file-width="2000" data-file-height="1013" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Archivo:Mediterranean-major-cities.png" class="internal" title="Aumentar"></a></div>Principales ciudades de la cuenca mediterránea.</div></div></div>
<p>El Mediterráneo se enfrenta a varios problemas derivados de las actividades humanas: la <a href="/wiki/Sobrepesca" title="Sobrepesca">sobrepesca</a>, las técnicas de pesca destructivas, el exceso de urbanismo en la franja litoral, la <a href="/wiki/Contaminaci%C3%B3n_marina" title="Contaminación marina">contaminación</a> y el <a href="/wiki/Calentamiento_global" title="Calentamiento global">calentamiento global</a>.<sup id="cite_ref-36" class="reference separada"><a href="#cite_note-36"><span class="corchete-llamada">[</span>36<span class="corchete-llamada">]</span></a></sup>&#8203; La sobrepesca actual es más del doble que hace cincuenta años aunque gracias a los esfuerzos ha bajado hasta la cifra actual de millón y medio de toneladas todavía demasiado alta para lo que este mar puede soportar. Este mar, por ser semicerrado, es muy sensible a la contaminación que produce la mala depuración de vertidos en los ríos, lo que significa una grave amenaza. El turismo mediterráneo suele acudir a zonas insuficientemente preparadas para aguantarlo y los vertidos no son tratados, además las obras erosionan las costas. También este acude a zonas de riqueza ecológica destruyendo el hábitat de especies en peligro (como <a href="/wiki/Foca_monje" class="mw-redirect" title="Foca monje">focas</a> y <a href="/wiki/Tortuga_marina" class="mw-redirect" title="Tortuga marina">tortugas</a>).
</p><p>Por otra parte el <a href="/wiki/Cambio_clim%C3%A1tico" title="Cambio climático">cambio climático</a> afecta aumentando la <a href="/wiki/Salinidad" title="Salinidad">salinidad</a> del mar y su temperatura. Según Vargas Yánez y colaboradores,<sup id="cite_ref-37" class="reference separada"><a href="#cite_note-37"><span class="corchete-llamada">[</span>37<span class="corchete-llamada">]</span></a></sup>&#8203; que han analizado datos desde 1948, el aumento medio de la temperatura superficial entre 1948 y 2005 del mar oscila entre 0,12&#160;°C y 0,5&#160;°C a lo largo del litoral mediterráneo; en profundidades intermedias (200 m a 600 m) la temperatura aumentó desde 1948 hasta 2000 entre 0,05&#160;°C y 0,2&#160;°C, y la salinidad se incrementó entre 0,03 y 0,09. En las capas profundas (1000 m a 2000 m) el aumento de temperatura osciló entre 0,03&#160;°C y 0,1&#160;°C y el de salinidad entre 0,05 y 0,06. El aumento de temperatura de las capas intermedias y profundas puede parecer pequeño, pero hay que tener en cuenta el alto <a href="/wiki/Calor_espec%C3%ADfico" title="Calor específico">calor específico</a> del mar, por lo que incrementos pequeños de temperatura requieren que el mar absorba enormes cantidades de calor. El aumento de salinidad refleja la disminución de las precipitaciones en el Mediterráneo, así como la disminución del aporte de los ríos debido a las obras hidráulicas llevadas a cabo en sus cauces. Desde mediados de la década de 1990, se ha observado un acusado aumento del nivel del mar de entre 2,5 mm/año y 10 mm/año, causado en parte por el aumento de la temperatura y en parte por el incremento del volumen de agua producido por el deshielo de los casquetes polares;
</p>
<div class="thumb tright"><div class="thumbinner" style="width:222px;"><a href="/wiki/Archivo:Posidonia_2_Alberto_Romeo.jpg" class="image"><img alt="" src="//upload.wikimedia.org/wikipedia/commons/thumb/e/e1/Posidonia_2_Alberto_Romeo.jpg/220px-Posidonia_2_Alberto_Romeo.jpg" decoding="async" width="220" height="165" class="thumbimage" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/e/e1/Posidonia_2_Alberto_Romeo.jpg/330px-Posidonia_2_Alberto_Romeo.jpg 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/e/e1/Posidonia_2_Alberto_Romeo.jpg/440px-Posidonia_2_Alberto_Romeo.jpg 2x" data-file-width="1280" data-file-height="960" /></a>  <div class="thumbcaption"><div class="magnify"><a href="/wiki/Archivo:Posidonia_2_Alberto_Romeo.jpg" class="internal" title="Aumentar"></a></div>Pradera de <i><a href="/wiki/Posidonia_oce%C3%A1nica" class="mw-redirect" title="Posidonia oceánica">Posidonia oceánica</a></i>.</div></div></div><p>Otras amenazas significativas son la desaparición de las praderas submarinas de <i><a href="/wiki/Posidonia_(planta)" class="mw-redirect" title="Posidonia (planta)">Posidonia</a></i>, una fanerógama marina que fundamenta gran parte de la <a href="/wiki/Diversidad_biol%C3%B3gica" class="mw-redirect" title="Diversidad biológica">diversidad biológica</a> mediterránea y las plagas de <a href="/wiki/Medusa_(animal)" class="mw-redirect" title="Medusa (animal)">medusas</a> lo que destruye el frágil ecosistema.
</p><h3><span id="Conservaci.C3.B3n"></span><span class="mw-headline" id="Conservación">Conservación</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mar_Mediterr%C3%A1neo&amp;action=edit&amp;section=15" title="Editar sección: Conservación">editar</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>La <a href="/wiki/Derecho_de_la_Uni%C3%B3n_Europea" title="Derecho de la Unión Europea">legislación europea</a> en materia de medio ambiente es prolija en la definición, mediante la <a href="/wiki/Directiva_92/43/CEE" class="mw-redirect" title="Directiva 92/43/CEE">directiva 92/43/CEE</a> (directiva Hábitat) y su ampliación con la <a href="/w/index.php?title=Directiva_97/62/CEE&amp;action=edit&amp;redlink=1" class="new" title="Directiva 97/62/CEE (aún no redactado)">directiva 97/62/CEE</a>, de medidas de protección del medio ambiente y de espacios diseñados para tal fin, como son los <a href="/wiki/Lugar_de_Importancia_Comunitaria" class="mw-redirect" title="Lugar de Importancia Comunitaria">lugares de interés comunitario</a> o LICs o las <a href="/wiki/Zona_de_Especial_Conservaci%C3%B3n" class="mw-redirect" title="Zona de Especial Conservación">zonas especiales de conservación</a> o ZECs, además de mediante otras directivas como la de aves, la <a href="/wiki/Directiva_79/409/CEE" class="mw-redirect" title="Directiva 79/409/CEE">directiva 79/409/CEE</a>, que define los entornos ZEPAs o <a href="/wiki/Zona_de_Especial_Protecci%C3%B3n_para_Aves" class="mw-redirect" title="Zona de Especial Protección para Aves">Zonas de Especial Protección para Aves</a>; ambos interrelacionados mediante la <a href="/wiki/Red_Natura_2000" title="Red Natura 2000">Red Natura 2000</a>.
</p><p>En materia de convenios suscritos entre estados, estos carentes de obligaciones legales en cuanto a cumplimiento, afectan al mar Mediterráneo de distinta forma: el convenio sobre la diversidad biológica de Río de Janeiro; el <a href="/wiki/CITES" title="CITES">convenio sobre el comercio de especies amenazadas de fauna y flora silvestres</a> (CITES por sus siglas en inglés); el Convenio sobre la conservación de la vida silvestre en Europa y de los hábitats naturales; el convenio sobre la conservación e las especies migratorias de la fauna silvestre; el convenio de Ramsar, sobre la conservación de <a href="/wiki/Humedal" title="Humedal">humedales</a>; y el convenio para la protección del Mar Mediterráneo (<a href="/wiki/Convenio_de_Barcelona" class="mw-redirect" title="Convenio de Barcelona">Convenio de Barcelona</a>). Dentro de este último, España firmó en 1995 el “Protocolo sobre Zonas Especialmente Protegidas y la Diversidad Biológica en el Mediterráneo” y adoptó un año después, en Montecarlo, sus anexos. Según este Protocolo cada Parte Contratante debe establecer Zonas Especialmente Protegidas de Importancia para el Mediterráneo (ZEPIM) en las zonas marinas y costeras sometidas a su soberanía y jurisdicción.<sup id="cite_ref-38" class="reference separada"><a href="#cite_note-38"><span class="corchete-llamada">[</span>38<span class="corchete-llamada">]</span></a></sup>&#8203;
</p><p>A nivel estatal y regional, existe numerosa legislación que determina las actuaciones a nivel local; no obstante, el carácter normativo de las directivas europeas provoca una cierta homogeneidad en ésta, que supone, en muchos casos, una mera traslación de aquella.
</p><p>En el marco de la gestión de la diversidad marina, algunos autores califican negativamente la situación jurídica del mar Mediterráneo:
<style data-mw-deduplicate="TemplateStyles:r119797252">.mw-parser-output .flexquote{display:flex;flex-direction:column;background-color:#F9F9F9;border-left:3px solid #c8ccd1;font-size:90%;margin:1em 4em;padding:.4em .8em}.mw-parser-output .flexquote>.flex{display:flex;flex-direction:row}.mw-parser-output .flexquote>.flex>.quote{width:100%}.mw-parser-output .flexquote>.flex>.separator{border-left:1px solid #c8ccd1;border-top:1px solid #c8ccd1;margin:.4em .8em}.mw-parser-output .flexquote>.cite{text-align:right}@media all and (max-width:600px){.mw-parser-output .flexquote>.flex{flex-direction:column}}</style>
</p>
<blockquote class="flexquote">
 <div class="flex">
  <div class="quote">(...) La situación jurídica del mar Mediterráneo, que se adivina relativamente compleja, convierte el
<p>régimen de conservación de la biodiversidad marina en una normativa fragmentada e inadecuada.
Solamente con el fomento de mayores esfuerzos de coordinación, sería posible el desarrollo de un
</p>
régimen jurídico integrado de conservación sostenible de la biodiversidad marina y pesquera (...)</div>
 </div><div class="cite">Claudiane Chevalier<sup id="cite_ref-chevalier_39-0" class="reference separada"><a href="#cite_note-chevalier-39"><span class="corchete-llamada">[</span>39<span class="corchete-llamada">]</span></a></sup>&#8203;</div>
</blockquote>
<h2><span id="Datos_de_inter.C3.A9s"></span><span class="mw-headline" id="Datos_de_interés">Datos de interés</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Mar_Mediterr%C3%A1neo&amp;action=edit&amp;section=16" title="Editar sección: Datos de interés">editar</a><span class="mw-editsection-bracket">]</span></span></h2>
<ul><li>Superficie: 2,51 millones de km².</li>
<li>Dimensiones: 4000 km de este a oeste; <span style="white-space:nowrap">46 000 km</span> de litoral; 800 km de norte a sur.</li>
<li>Profundidad: media: 1370 m, máxima: 5210 m (<a href="/wiki/Fosa_de_Calipso" title="Fosa de Calipso">Fosa de Calipso</a>, en Grecia).</li>

<li>Aporte pesquero: aproximadamente el 2&#160;% del total mundial.</li>
<li>Salinidad media: entre 36 a 38 gramos por litro.</li>
<li>Ríos más importantes: <a href="/wiki/Adigio" title="Adigio">Adigio</a>, <a href="/wiki/Drin" class="mw-redirect" title="Drin">Drin</a>, <a href="/wiki/Ebro" title="Ebro">Ebro</a>, <a href="/wiki/J%C3%BAcar" title="Júcar">Júcar</a>, <a href="/wiki/R%C3%ADo_Maritsa" title="Río Maritsa">Maritza</a>, <a href="/wiki/R%C3%ADo_Menderes" title="Río Menderes">Menderes</a>, <a href="/wiki/R%C3%ADo_Muluya" title="Río Muluya">Muluya</a>. <a href="/wiki/Nilo" title="Nilo">Nilo</a>, <a href="/wiki/Neretva" class="mw-redirect" title="Neretva">Neretva</a>, <a href="/wiki/Orontes" title="Orontes">Orontes</a>, <a href="/wiki/Po" title="Po">Po</a>, <a href="/wiki/R%C3%B3dano" title="Ródano">Ródano</a>, <a href="/wiki/R%C3%ADo_Segura" title="Río Segura">Segura</a>, <a href="/wiki/T%C3%ADber" title="Tíber">Tíber</a>, <a href="/wiki/Vardar" class="mw-redirect" title="Vardar">Vardar</a>.</li></ul>

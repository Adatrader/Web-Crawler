<dl><dt>1</dt>
<dd>Página de un <a href="/wiki/libro" title="libro">libro</a> donde, generalmente, aparecen el nombre del autor, el título del libro y la editorial.</dd></dl>
<dl><dt>2</dt>
<dd><a href="/wiki/ornamento#Español" title="ornamento">Ornamento</a> arquitectónico que se hace en las fachadas principales de los edificios <a href="/w/index.php?title=fastuoso&amp;action=edit&amp;redlink=1" class="new" title="fastuoso (la página no existe)">fastuosos</a>.</dd></dl>
<dl><dt>3</dt>
<dd>En un periódico o una revista, primera página.
<ul><li><b>Antónimo:</b> <span lang="es" style="" class=""><a href="/wiki/contraportada#Español" title="contraportada">contraportada</a></span>.</li></ul></dd></dl>
<dl><dt>4</dt>
<dd><a href="/wiki/cubierta#Español" title="cubierta">Cubierta</a> delantera de un libro u otra publicación.</dd></dl>
<dl><dt>5</dt>
<dd>En el arte de la <a href="/wiki/seda" title="seda">seda</a>, distribución que se hace de cierta cantidad de hilos para formar la <a href="/wiki/urdimbre" title="urdimbre">urdimbre</a>.</dd></dl>

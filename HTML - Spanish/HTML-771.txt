<p><b>España</b>, también denominado <b>Reino de España</b>, es un <a href="/wiki/Estado" title="Estado">país soberano</a>, miembro de la <a href="/wiki/Uni%C3%B3n_Europea" title="Unión Europea">Unión Europea</a>, constituido en <a href="/wiki/Estado_social" title="Estado social">Estado social</a> y <a href="/wiki/Democracia" title="Democracia">democrático</a> de <a href="/wiki/Estado_de_Derecho" class="mw-redirect" title="Estado de Derecho">Derecho</a> y cuya forma de gobierno es la <a href="/wiki/Monarqu%C3%ADa_parlamentaria" title="Monarquía parlamentaria">monarquía parlamentaria</a>. Su territorio, con capital en <a href="/wiki/Madrid" title="Madrid">Madrid</a>, ocupa la mayor parte de la <a href="/wiki/Pen%C3%ADnsula_Ib%C3%A9rica" class="mw-redirect" title="Península Ibérica">península Ibérica</a>, al que se añaden los <a href="/wiki/Archipi%C3%A9lago" title="Archipiélago">archipiélagos</a> de las <a href="/wiki/Islas_Baleares" title="Islas Baleares">islas Baleares</a> (en el <a href="/wiki/Mar_Mediterr%C3%A1neo" title="Mar Mediterráneo">mar Mediterráneo</a> occidental) y de las <a href="/wiki/Canarias" title="Canarias">islas Canarias</a> (en el <a href="/wiki/Oc%C3%A9ano_Atl%C3%A1ntico" title="Océano Atlántico">océano Atlántico</a> nororiental), así como en el norte del <a href="/wiki/%C3%81frica" title="África">continente africano</a> las <a href="/wiki/Ciudad_aut%C3%B3noma" title="Ciudad autónoma">ciudades autónomas</a> de <a href="/wiki/Ceuta" title="Ceuta">Ceuta</a> y <a href="/wiki/Melilla" title="Melilla">Melilla</a>, además de los distritos y posesiones menores de las <a href="/wiki/Islas_Chafarinas" title="Islas Chafarinas">islas Chafarinas</a>, el <a href="/wiki/Pe%C3%B1%C3%B3n_de_V%C3%A9lez_de_la_Gomera" title="Peñón de Vélez de la Gomera">peñón de Vélez de la Gomera</a> y el <a href="/wiki/Pe%C3%B1%C3%B3n_de_Alhucemas" title="Peñón de Alhucemas">peñón de Alhucemas</a>. El <a href="/wiki/Municipio" title="Municipio">municipio</a> de <a href="/wiki/Llivia" title="Llivia">Llivia</a>, rodeado totalmente por territorio francés, en los <a href="/wiki/Pirineos" title="Pirineos">Pirineos</a>, completa el conjunto de territorios junto con la <a href="/wiki/Isla_de_Albor%C3%A1n" title="Isla de Alborán">isla de Alborán</a>, las <a href="/wiki/Islas_Columbretes" class="mw-redirect" title="Islas Columbretes">islas Columbretes</a> y una serie de <a href="/wiki/Anexo:Islas_de_Espa%C3%B1a" title="Anexo:Islas de España">islas e islotes</a> frente a sus propias costas.
</p><p>Tiene una extensión de 504.645 <a href="/wiki/Kil%C3%B3metro_cuadrado" title="Kilómetro cuadrado">km²</a>, siendo el cuarto país más extenso del continente, tras <a href="/wiki/Rusia" title="Rusia">Rusia</a>, <a href="/wiki/Ucrania" title="Ucrania">Ucrania</a> y <a href="/wiki/Francia" title="Francia">Francia</a>. Con una altitud media de 650 metros es uno de los países más montañosos de <a href="/wiki/Europa" title="Europa">Europa</a>. Su población es de 47.190.493 habitantes, según datos del <a href="/wiki/Padr%C3%B3n_municipal_de_habitantes" title="Padrón municipal de habitantes">padrón municipal</a> de <a href="/wiki/2011" title="2011">2011</a>.

<p><b>Andrés Manuel del Río Fernández</b> nació el <a href="/wiki/10_de_noviembre" title="10 de noviembre">10 de noviembre</a> de <a href="/wiki/1764" title="1764">1764</a> en <a href="/wiki/Madrid" title="Madrid">Madrid</a> (<a href="/wiki/Espa%C3%B1a" title="España">España</a>) y falleció el <a href="/wiki/23_de_marzo" title="23 de marzo">23 de marzo</a> de <a href="/wiki/1849" title="1849">1849</a> en la <a href="/wiki/Ciudad_de_M%C3%A9xico" title="Ciudad de México">Ciudad de México</a>. <a href="/wiki/Cient%C3%ADfico" title="Científico">Científico</a> y <a href="/wiki/Naturalista" title="Naturalista">naturalista</a>, fue el descubridor del <a href="/wiki/Elemento_qu%C3%ADmico" title="Elemento químico">elemento químico</a> <a href="/wiki/Vanadio" title="Vanadio">vanadio</a>.
</p><p>Estudió <a href="/wiki/Qu%C3%ADmica_anal%C3%ADtica" title="Química analítica">química analítica</a> y <a href="/wiki/Metalurgia" title="Metalurgia">metalurgia</a> en su país natal. Realizó estudios de filosofía, teología y literatura y se graduó como bachiller en 1780 en la <a href="/wiki/Universidad_de_Alcal%C3%A1_(hist%C3%B3rica)" title="Universidad de Alcalá (histórica)">Universidad de Alcalá de Henares</a> y posteriormente ingresó en la <a href="/wiki/Almad%C3%A9n" title="Almadén">Escuela de Minería de Almadén</a>, en España. <b><a href="/wiki/Andr%C3%A9s_Manuel_del_R%C3%ADo" title="Andrés Manuel del Río">Leer más...</a></b>

<p>Si quieres colaborar con los artículos relacionados con <a href="/wiki/Espa%C3%B1a" title="España">España</a> en <a href="/wiki/Wikipedia" title="Wikipedia">Wikipedia</a> puedes: 
</p>

<li>Mejorar los artículos que requieran corregir conceptos, errores ortográficos, coherencia entre párrafos, etc.</li>
<li>Ampliar <a href="/wiki/Wikipedia:Encontrar_y_arreglar_un_esbozo" class="mw-redirect" title="Wikipedia:Encontrar y arreglar un esbozo">esbozos</a> y miniesbozos.</li>
<li>Mejorar los artículos a los que les falte poco para que se conviertan en <a href="/wiki/Wikipedia:Qu%C3%A9_es_un_art%C3%ADculo_bueno" title="Wikipedia:Qué es un artículo bueno">buenos</a> o <a href="/wiki/Wikipedia:Qu%C3%A9_es_un_art%C3%ADculo_destacado" title="Wikipedia:Qué es un artículo destacado">destacados</a></li>
<li>Subir imágenes a <a class="external text" href="https://commons.wikimedia.org/wiki/Spain">Commons</a>.</li>
<li>Traducir artículos de otras wikipedias</li></ul>

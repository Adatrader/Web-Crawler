<p><b>What an edit request IS NOT for:</b><br />
<a href="/wiki/File:X_mark.svg" class="image"><img alt="X mark.svg" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/a2/X_mark.svg/15px-X_mark.svg.png" decoding="async" width="15" height="17" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/a2/X_mark.svg/23px-X_mark.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/a2/X_mark.svg/30px-X_mark.svg.png 2x" data-file-width="525" data-file-height="600" /></a> making test edits: use the <a href="/wiki/Wikipedia:SANDBOX" class="mw-redirect" title="Wikipedia:SANDBOX">sandbox</a><br />
<a href="/wiki/File:X_mark.svg" class="image"><img alt="X mark.svg" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/a2/X_mark.svg/15px-X_mark.svg.png" decoding="async" width="15" height="17" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/a2/X_mark.svg/23px-X_mark.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/a2/X_mark.svg/30px-X_mark.svg.png 2x" data-file-width="525" data-file-height="600" /></a> asking for help: go to the <a href="/wiki/Wikipedia:HELPDESK" class="mw-redirect" title="Wikipedia:HELPDESK">Help Desk</a><br />
<a href="/wiki/File:X_mark.svg" class="image"><img alt="X mark.svg" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/a2/X_mark.svg/15px-X_mark.svg.png" decoding="async" width="15" height="17" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/a2/X_mark.svg/23px-X_mark.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/a2/X_mark.svg/30px-X_mark.svg.png 2x" data-file-width="525" data-file-height="600" /></a> making a comment or starting a discussion: go to the <a class="mw-selflink selflink">talk page</a><br />
<a href="/wiki/File:X_mark.svg" class="image"><img alt="X mark.svg" src="//upload.wikimedia.org/wikipedia/commons/thumb/a/a2/X_mark.svg/15px-X_mark.svg.png" decoding="async" width="15" height="17" srcset="//upload.wikimedia.org/wikipedia/commons/thumb/a/a2/X_mark.svg/23px-X_mark.svg.png 1.5x, //upload.wikimedia.org/wikipedia/commons/thumb/a/a2/X_mark.svg/30px-X_mark.svg.png 2x" data-file-width="525" data-file-height="600" /></a> requesting unprotection: go to <a href="/wiki/Wikipedia:RFPP" class="mw-redirect" title="Wikipedia:RFPP">RFPP</a> <small>(unprotection requests should take into account the original protection reason - see the <a class="external text" href="https://en.wikipedia.org/w/index.php?title=Special:Log&amp;type=protect&amp;page=Florida">protection log</a>)</small><br /><br />
</p>

<p><b>What an edit request IS for:</b><br />
<a href="/wiki/File:Yes_check.svg" class="image"><img alt="Yes check.svg" src="//upload.wikimedia.org/wikipedia/en/thumb/f/fb/Yes_check.svg/15px-Yes_check.svg.png" decoding="async" width="15" height="15" srcset="//upload.wikimedia.org/wikipedia/en/thumb/f/fb/Yes_check.svg/23px-Yes_check.svg.png 1.5x, //upload.wikimedia.org/wikipedia/en/thumb/f/fb/Yes_check.svg/30px-Yes_check.svg.png 2x" data-file-width="600" data-file-height="600" /></a> requesting straight-forward changes like grammar, spelling, formatting<br />
<a href="/wiki/File:Yes_check.svg" class="image"><img alt="Yes check.svg" src="//upload.wikimedia.org/wikipedia/en/thumb/f/fb/Yes_check.svg/15px-Yes_check.svg.png" decoding="async" width="15" height="15" srcset="//upload.wikimedia.org/wikipedia/en/thumb/f/fb/Yes_check.svg/23px-Yes_check.svg.png 1.5x, //upload.wikimedia.org/wikipedia/en/thumb/f/fb/Yes_check.svg/30px-Yes_check.svg.png 2x" data-file-width="600" data-file-height="600" /></a> requesting more substantive changes, if they are justified with reference to <a href="/wiki/Wikipedia:RS" class="mw-redirect" title="Wikipedia:RS">reliable sources</a>, and not the subject of ongoing discussion
</p>

<p><span style="color:red;"><i><b>Follow the instructions below to submit an edit request</b></i></span> - a request to have someone edit the article for you. Please be clear and precise in explaining and justifying the change. An <a href="/wiki/Wikipedia:User_access_levels#Autoconfirmed_users" title="Wikipedia:User access levels">established user</a> may then make the change on your behalf.
</p>
<ol><li><b>Please leave the existing codes intact.</b> Write your request below the <style data-mw-deduplicate="TemplateStyles:r886049734">.mw-parser-output .monospaced{font-family:monospace,monospace}</style><span class="monospaced">&lt;!-- State UNAMBIGUOUSLY your suggested changes...</span> line and above the <link rel="mw-deduplicated-inline-style" href="mw-data:TemplateStyles:r886049734"/><span class="monospaced">&lt;!-- Write your request ABOVE this line...</span> line.</li>
<li><b>Please provide a <i>specific</i> description of the edit request</b>, that is, specific text that should be removed and a verbatim copy of the text that should replace it. "Please change X" is not helpful and will often be rejected; the request should be of the form "please change X to Y because...".</li>
<li><b>Please don't copy the entire article into the request.</b> Only copy the part you're changing. If you copy the entire article into the request, you'll break navigation on the talk page, and another editor may remove your entire request.</li>
<li><b>Please <u>provide reliable sources</u> if appropriate.</b> All information in Wikipedia articles should be <a href="/wiki/Wikipedia:V" class="mw-redirect" title="Wikipedia:V">verifiable</a> from <a href="/wiki/Wikipedia:RS" class="mw-redirect" title="Wikipedia:RS">reliable sources</a> which are independent of the subject.</li></ol></td></tr></tbody></table>

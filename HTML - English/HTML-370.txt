<p>A template meant to facilitate the creation of a "Largest cities in country X" template. It accommodates either 10 or 20 entries, plus 4 pictures representing the 4 largest cities.
</p>
<h2><span class="mw-headline" id="Usage">Usage</span></h2>
<pre style="font-size:95%; overflow:auto;">{{Largest cities
| name        = &lt;!-- name of the template, e.g. Largest cities of Chile --&gt;

| country     = &lt;!-- the name of the country, e.g. Chile --&gt;
| kind        = &lt;!-- replace "cities and towns" in the title w/ anything, e.g. "conurbations" --&gt;
| stat_ref    = &lt;!-- source of the pop. figures, e.g. Instituto Nacional de Estadisticas de Chile or use {{URL| }} --&gt;
| list_by_pop = &lt;!-- link to the list of cities in the given country, if possible sorted by population --&gt;
| div_name    = &lt;!-- the name of the main administrative subdivision, e.g. Province, State, Region... --&gt;
| div_link    = &lt;!-- the template will automatically create a link for "div_name of country" (e.g. Provinces of Chile), if this doesn't work you can use this field --&gt;

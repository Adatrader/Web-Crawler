<p>Wikipedia takes <a href="/wiki/Wikipedia:Copyrights" title="Wikipedia:Copyrights">copyright law</a> very seriously. <a href="/wiki/Help:File_page" class="mw-redirect" title="Help:File page">File description pages</a> are <a href="/wiki/Wikipedia:Template_namespace" title="Wikipedia:Template namespace">tagged</a> with the <b>license</b> and the <b>source</b> of the file. This makes it as easy as possible for readers, Wikipedians, IP addresses and creators of derivative works to know what they can and can't do with the files in our encyclopedia.
</p>

<h2><span class="mw-headline" id="Guidelines">Guidelines</span></h2>
<ul><li>For a file to be considered "free" under Wikipedia's <a href="/wiki/Wikipedia:Image_use_policy" title="Wikipedia:Image use policy">Image use policy</a>, the license <b>must</b> permit both <b>commercial reuse</b> and <b>derivative works</b>.</li>
<li>Wikipedia (and all Wikimedia projects) strongly prefer "free" files. Where no free file exists, it is sometimes permissible to use a non-free (copyright-protected) file under the "<a href="/wiki/Wikipedia:Non-free_content" title="Wikipedia:Non-free content">fair use</a>" provision. Fair use, in the United States, is the legal right to use copyright-protected materials under certain conditions. Wikipedia has its own conditions that are more restrictive than what is legally permitted.</li>
<li>Ideally, most free files should be uploaded to <a href="https://commons.wikimedia.org/wiki/" class="extiw" title="commons:">Wikimedia Commons</a> instead of Wikipedia. (This enables them to be used for other Wikimedia projects in addition to Wikipedia.) Your account must be at least four days old before uploading is permitted. <a href="https://commons.wikimedia.org/wiki/Commons:Fair_use" class="extiw" title="commons:Commons:Fair use">Commons does not allow fair use material</a>, but non-free can still be used on the English Wikipedia under <a href="/wiki/Wikipedia:Copyrights#Non-free_materials_and_special_requirements" title="Wikipedia:Copyrights">certain conditions</a>.</li>
<li>All copyright and licensing tags should be put on a line of their own.</li>
<li>Along with a tag, specify the source or copyright holder information. Provide as much detail as possible.</li>
<li>If a file is not licensed under a standard license, please specify what the actual license states.</li>
<li>If you tag a file as requiring attribution, please specify who needs to be attributed.</li>
<li>If multiple categories apply to a file, add all that apply.</li></ul>

<h2><span class="mw-headline" id="For_image_creators">For image creators</span></h2>
<link rel="mw-deduplicated-inline-style" href="mw-data:TemplateStyles:r911143475"/><div class="module-shortcutanchordiv"><span id="WP:ICTIC"></span></div><div class="module-shortcutboxplain plainlist noprint" role="note"><div class="module-shortcutlist"><a href="/wiki/Wikipedia:Shortcut" title="Wikipedia:Shortcut">Shortcut</a></div><ul><li><span class="plainlinks"><a class="external text" href="https://en.wikipedia.org/w/index.php?title=Wikipedia:ICTIC&amp;redirect=no">WP:ICTIC</a></span></li></ul></div>
<p>If you are the creator of a file, you can choose any acceptable <a href="/wiki/Wikipedia:Image_use_policy#Free_licenses" title="Wikipedia:Image use policy">free license</a>. You can multi-license your file under different licenses, if you prefer. The license must <b>not</b> prevent <b>commercial reuse</b> or <b>derivative works</b>.

<h2><span class="mw-headline" id="Creating_new_tags">Creating new tags</span></h2>
<p>If you are uploading many files with the same source and license, you can create a new copyright tag. Please propose the tag on <a href="/wiki/Wikipedia_talk:File_copyright_tags" title="Wikipedia talk:File copyright tags">Wikipedia talk:File copyright tags</a>. If you are not familiar with how to create <a href="/wiki/Wikipedia:Template_namespace" title="Wikipedia:Template namespace">templates</a> which add images to <a href="/wiki/Wikipedia:Categorization" title="Wikipedia:Categorization">categories</a>, please ask for assistance.
</p><p>Each template should have a corresponding category that automatically contains all the files tagged with the template. On the description page for the category include the following:
</p>
<pre>{{File template notice|<b>name-of-the-tag</b>}}
</pre>
<p>where <b>name-of-the-tag</b> is replaced with the actual name of the tag. In addition, include the template in the category <a href="/wiki/Category:Wikipedia_file_copyright_templates" title="Category:Wikipedia file copyright templates">Wikipedia file copyright templates</a> by adding the following:
</p>
<pre>&lt;noinclude&gt;[[Category:Wikipedia file copyright templates|{{PAGENAME}}]]&lt;/noinclude&gt;
</pre>

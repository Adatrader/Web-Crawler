<p><b><span style="font-size: 120%;">Initial visibility:</span></b> currently defaults to <b><span style="font-size: 120%;">autocollapse</span></b>
</p><p>To set this template's initial visibility, the <code class="nowrap" style="">&#124;state&#61;</code> <a href="/wiki/Help:Template#Parameters" title="Help:Template">parameter</a> may be used:
</p>
<ul>
<li><code class="nowrap" style="">&#124;state&#61;collapsed</code>: <code>{{English official language clickable map|state=collapsed}}</code> to show the template collapsed, i.e., hidden apart from its title bar</li>
<li><code class="nowrap" style="">&#124;state&#61;expanded</code>: <code>{{English official language clickable map|state=expanded}}</code> to show the template expanded, i.e., fully visible</li>
<li><code class="nowrap" style="">&#124;state&#61;autocollapse</code>: <code>{{English official language clickable map|state=autocollapse}}</code>
<ul>
<li>shows the template collapsed to the title bar if there is a <code>{{<a href="/wiki/Template:Navbar" title="Template:Navbar">navbar</a>}}</code>, a <code>{{<a href="/wiki/Template:Sidebar" title="Template:Sidebar">sidebar</a>}}</code>, or some other table on the page with the collapsible attribute</li>
<li>shows the template in its expanded state if there are no other collapsible items on the page</li>
</ul></li></ul>
<p>If the <code class="nowrap" style="">&#124;state&#61;</code> parameter in the template on this page is not set, the template's initial visibility is taken from the <code class="nowrap" style="">&#124;default&#61;</code> parameter in the <i>Collapsible option</i> template. For the template on this page, that currently evaluates to <span style="font-size: 120%;"><b><code>autocollapse</code></b></span>.

Any and all help is very greatly appreciated!

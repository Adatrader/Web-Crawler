<td><i>This page is a short introduction to the Wikipedia's multilingual concepts.</i>
<ul><li>The main central place for the coordination of all the Wikimedia projects (in any language) is the <a href="/wiki/Wikipedia:Meta-Wiki" class="mw-redirect" title="Wikipedia:Meta-Wiki">Meta-Wiki</a> (located at <a href="https://meta.wikimedia.org/wiki/main_page" class="extiw" title="m:main page">meta.wikimedia.org</a>).</li>
<li>The <a href="/wiki/Wikipedia:Mailing_lists" title="Wikipedia:Mailing lists">mailing list</a> to coordinate Wikipedia across different languages is <code>Wikipedia-l</code>.</li>
<li>See also the English Wikipedia's <a href="/wiki/Wikipedia:Embassy" title="Wikipedia:Embassy">embassy</a>, a page for non-native speakers.</li></ul>

<p>The <a href="/wiki/Wikipedia:Wikipedia_community" class="mw-redirect" title="Wikipedia:Wikipedia community">Wikipedia community</a> is committed to including any and all languages for which there are <a href="/wiki/Wikipedia:Wikipedians" title="Wikipedia:Wikipedians">Wikipedians</a> willing to do the work. We are aware that many of the <a href="/wiki/Language_family" title="Language family">world's 6,500 languages</a> are not well-represented on computers or the web, and we are committed to working with language speakers and computing organizations to support as many languages as possible.
</p>

<p>Each language in Wikipedia currently has a separate set of user accounts. Links between articles in different languages are called <a href="/wiki/Help:Interlanguage_links" title="Help:Interlanguage links">interlanguage links</a> or (colloquially and a bit ambiguously) "<a href="/wiki/InterWiki" class="mw-redirect" title="InterWiki">interwiki</a>".
</p><p>The URL of the wikipedia for a given language is xx.wikipedia.org, where xx is the  2-letter language  code as per <a href="/wiki/ISO_639" title="ISO 639">ISO 639</a>. For languages without an ISO 639 2-letter language code, the 3-letter language code is used, or if that also does not exist, a custom 3-letter language code is made. (On the mailing list and in discussion, people often write <i>xx:</i> to mean <i>the xx-language wikipedia</i>, as in <i>I'm a regular on fr:</i>.)
</p>

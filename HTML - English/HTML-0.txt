<p>The license <a href="/wiki/Wikiversity" class="mw-redirect" title="Wikiversity">Wikiversity</a> uses grants free access to our <a href="/wiki/Learning_resource" title="Learning resource">content</a> in the same sense as <a href="https://en.wikipedia.org/wiki/free_software" class="extiw" title="w:free software">free software</a> is licensed freely. This principle is known as <b><a href="https://en.wikipedia.org/wiki/copyleft" class="extiw" title="w:copyleft">copyleft</a></b>. That is to say, Wikiversity content can be copied, modified, and redistributed, either commercially or noncommercially, <i>so long as</i> the new version grants the same freedoms to others and acknowledges the authors of the Wikiversity content used (a direct link back to the article satisfies our author credit requirement).  Wikiversity <a href="/wiki/Learning_resource" title="Learning resource">learning resources</a> therefore will remain free forever and can be used by anybody subject to certain restrictions, most of which serve to ensure that freedom.
</p><p>To fulfill the above goals, the text contained in Wikiversity is licensed to the public under the <a rel="nofollow" class="external text" href="https://creativecommons.org/licenses/by-sa/4.0/">Creative Commons Attribution/Share-Alike License</a> and <b><a href="/wiki/Wikiversity:GNU_Free_Documentation_License" title="Wikiversity:GNU Free Documentation License">GNU Free Documentation License</a></b> (GFDL). Please see <a rel="nofollow" class="external text" href="https://wikimediafoundation.org/wiki/Terms_of_Use">Terms of Use</a>.
</p><p>For material whose copyright is in question see <a href="/wiki/Wikiversity:Copyright_issues" title="Wikiversity:Copyright issues">Wikiversity:Copyright issues</a>. This also includes information on how to deal with such material.
</p>
<h2><span class="mw-headline" id="Copyright_and_Media_Uploads">Copyright and Media Uploads</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Wikiversity:Copyrights&amp;action=edit&amp;section=1" title="Edit section: Copyright and Media Uploads">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>According to <a href="https://foundation.wikimedia.org/wiki/Resolution:Licensing_policy" class="extiw" title="foundation:Resolution:Licensing policy">Foundation policy</a>, uploads are required to have a <a href="/wiki/Wikiversity:License_tags" title="Wikiversity:License tags">license template</a> and source. To add this information after upload, please edit the image description page directly (click the <i>edit this page</i> tab); uploading a new version of the media <i>will not</i> update the license template, nor the source information.
</p>

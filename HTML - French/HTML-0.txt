<p>
<i><b><span class="lang-la" lang="la">Ex nihilo</span></b></i> est une expression <a href="/wiki/Latin" title="Latin">latine</a> signifiant «&#160;à partir de <a href="/wiki/Rien" title="Rien">rien</a>&#160;». Elle est souvent utilisée en conjonction avec un terme exprimant une idée de <a href="/wiki/Cr%C3%A9ativit%C3%A9" title="Créativité">création</a>, comme dans «&#160;création <i>ex nihilo</i>&#160;», signifiant littéralement «&#160;création à partir de rien&#160;», par opposition à une «&#160;<i><span class="lang-la" lang="la">creation ex materia</span></i>&#160;», création à partir d'un matériau ou d'un substrat préexistant.
</p><p>Cette expression apparaît dans la formulation <span class="lang-la" lang="la">«&#160;ex nihilo nihil fit&#160;»</span>, «&#160;rien ne vient de rien&#160;», et se retrouve en science avec le <a href="/wiki/Premier_principe_de_la_thermodynamique" title="Premier principe de la thermodynamique">premier principe de la thermodynamique</a> (conservation de l'énergie).
</p><p>On parle aussi de création <i>ex nihilo</i> pour la <a href="/wiki/Cr%C3%A9ation_mon%C3%A9taire" title="Création monétaire">création monétaire</a> <a href="/wiki/Monnaie_fiduciaire" title="Monnaie fiduciaire">fiduciaire</a> (par la <a href="/wiki/Banque_centrale" title="Banque centrale">banque centrale</a>) ou scripturale (par les <a href="/wiki/Banque" title="Banque">banques</a> commerciales)&#160;; le <a href="/wiki/Cr%C3%A9dit" title="Crédit">crédit</a> est une création monétaire <i>ex nihilo</i><sup id="cite_ref-1" class="reference"><a href="#cite_note-1"><span class="cite_crochet">[</span>1<span class="cite_crochet">]</span></a></sup>.
</p><p>Exemple&#160;: Aujourd'hui beaucoup de stations balnéaires sont créées ex nihilo.
</p>
<h2><span class="mw-headline" id="Sens_moderne">Sens moderne</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Ex_nihilo&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Modifier la section : Sens moderne">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Ex_nihilo&amp;action=edit&amp;section=1" title="Modifier la section : Sens moderne">modifier le code</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Cette expression a pour équivalent <a href="/wiki/Anglais" title="Anglais">anglais</a> <i><a href="https://fr.wiktionary.org/wiki/en:from_scratch" class="extiw" title="wikt:en:from scratch">from scratch</a></i>, employée notamment dans le jargon de la <a href="/wiki/Programmation_informatique" title="Programmation informatique">programmation informatique</a>, comme dans <a href="/wiki/Linux_From_Scratch" title="Linux From Scratch">Linux From Scratch</a>.
</p><p>Le secteur touristique utilise également cette expression dans la <a rel="nofollow" class="external text" href="https://tel.archives-ouvertes.fr/tel-00683463/">désignation de tourisme hors sol</a> par exemple des <a href="/wiki/Station_de_sports_d%27hiver" title="Station de sports d&#39;hiver">stations de ski ex nihilo</a> 
</p>

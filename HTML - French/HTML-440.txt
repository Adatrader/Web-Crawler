<td colspan="2"><a href="/wiki/Annexe:Prononciation/fran%C3%A7ais" title="Annexe:Prononciation/français"><span class="API" title="Prononciation API">\ʁə.du.blə.mɑ̃\</span></a>

<p><b>redoublement</b> <a href="/wiki/Annexe:Prononciation/fran%C3%A7ais" title="Annexe:Prononciation/français"><span class="API" title="Prononciation API">\ʁə.du.blə.mɑ̃\</span></a> <span class="ligne-de-forme"><i>masculin</i></span>
</p>
<ol><li><a href="/wiki/accroissement" title="accroissement">Accroissement</a>, <a href="/wiki/augmentation" title="augmentation">augmentation</a> <a href="/wiki/consid%C3%A9rable" title="considérable">considérable</a>.
<ul><li><i>Ainsi protégés, ils marchèrent rapidement, car le vent, les prenant de dos, les poussa avec une extrême violence, et, par certains <b>redoublements</b> de la rafale, on peut dire qu’ils allaient plus vite qu’ils ne le voulaient.</i> <span class="sources"><span class="tiret">—&#160;</span>(<a href="https://fr.wikipedia.org/wiki/Jules_Verne" class="extiw" title="w:Jules Verne">Jules&#160;<span class="petites_capitales" style="font-variant: small-caps">Verne</span></a>, <i><a href="https://fr.wikipedia.org/wiki/Le_Pays_des_fourrures" class="extiw" title="w:Le Pays des fourrures">Le Pays des fourrures</a></i>, <a href="https://fr.wikipedia.org/wiki/Pierre-Jules_Hetzel" class="extiw" title="w:Pierre-Jules Hetzel">J.&#160;<span class="petites_capitales" style="font-variant: small-caps">Hetzel</span> et&#160;C<sup style="font-size:83.33%;line-height:1">ie</sup></a>, Paris, <a href="https://fr.wikisource.org/wiki/Le_Pays_des_fourrures" class="extiw" title="s:Le Pays des fourrures">1873</a>)</span></li>
<li><i>Chaque étranger qui arrivait et mettait pied à terre, excitait dans la foule un <b>redoublement</b> de curiosité </i>[…] <span class="sources"><span class="tiret">—&#160;</span>(<a href="https://fr.wikipedia.org/wiki/Julie_de_Qu%C3%A9rangal" class="extiw" title="w:Julie de Quérangal">Julie de&#160;<span class="petites_capitales" style="font-variant: small-caps">Quérangal</span></a>, <i><a href="https://fr.wikisource.org/wiki/Philippe_de_Morvelle" class="extiw" title="s:Philippe de Morvelle">Philippe de Morvelle</a></i>, Revue des Deux Mondes, t. 2, 4, 1833)</span></li></ul></li>
<li><span class="term" id="médecine"><i>(<span class="texte">Médecine</span>)</i></span> Augmentation <a href="/wiki/p%C3%A9riodique" title="périodique">périodique</a> ou <a href="/wiki/irr%C3%A9gulier" title="irrégulier">irrégulière</a> dans l’<a href="/wiki/intensit%C3%A9" title="intensité">intensité</a> des <a href="/wiki/sympt%C3%B4me" title="symptôme">symptômes</a> d’une <a href="/wiki/maladie" title="maladie">maladie</a>, et <a href="/wiki/particuli%C3%A8rement" title="particulièrement">particulièrement</a> des <a href="/wiki/fi%C3%A8vre" title="fièvre">fièvres</a>.
<ul><li><i>Un <b>redoublement</b> de fièvre. Avoir une fièvre continue avec des <b>redoublements</b>.</i></li></ul></li>
<li><span class="term" id="linguistique"><i>(<span class="texte">Linguistique</span>)</i></span> <a href="/wiki/d%C3%A9rivation" title="dérivation">Dérivation</a> par la répétition complète ou partielle d’un mot ou d’un de ses morphèmes, pour l’emphase, pour le pluriel, etc. Par exemple, <i><a href="/wiki/foufou" title="foufou">foufou</a></i> est créé par redoublement de <i><a href="/wiki/fou" title="fou">fou</a></i>. <a href="/wiki/r%C3%A9duplication" title="réduplication">Réduplication</a>.
<ul><li><i>Le pluriel par réduplication ou <b>redoublement</b> peut n’être pas seul à marquer la pluralité.</i> <span class="sources"><span class="tiret">—&#160;</span>(Claude Hagège, <i>Dictionnaire amoureux des Langues</i>, Plon/Odile Jacob, 2009, page 606, <a href="/wiki/Sp%C3%A9cial:Ouvrages_de_r%C3%A9f%C3%A9rence/9782259204095" class="internal mw-magiclink-isbn">ISBN 978-2-259-20409-5</a>)</span></li></ul>
<ol><li><span class="term"><i>(<span class="texte">Grammaire grecque</span>)</i></span> <a href="/wiki/r%C3%A9p%C3%A9tition" title="répétition">Répétition</a> de la <a href="/wiki/consonne" title="consonne">consonne</a> <a href="/wiki/initial" title="initial">initiale</a> du <a href="/wiki/radical" title="radical">radical</a> <a href="/wiki/devant" title="devant">devant</a> l’<a href="/wiki/augment" title="augment">augment</a>, au <a href="/wiki/parfait" title="parfait">parfait</a> des <a href="/wiki/verbe" title="verbe">verbes</a>.</li></ol></li>
<li><span class="term" id="éducation"><i>(<span class="texte">Éducation</span>)</i></span> <a href="/wiki/fait" title="fait">Fait</a> pour un <a href="/wiki/%C3%A9l%C3%A8ve" title="élève">élève</a> de reprendre le même <a href="/wiki/niveau" title="niveau">niveau</a>.
<ul><li><i>Dans l’idéal, le <b>redoublement</b> de classe devrait être exceptionnel.</i></li></ul></li></ol>

<h2><span id="Les_br.C3.A8ves"></span><span class="mw-headline" id="Les_brèves">Les brèves</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Aide:Comment_ajouter_du_contenu_avec_un_minimum_d%27effort&amp;veaction=edit&amp;section=1" class="mw-editsection-visualeditor" title="Modifier la section : Les brèves">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Aide:Comment_ajouter_du_contenu_avec_un_minimum_d%27effort&amp;action=edit&amp;section=1" title="Modifier la section : Les brèves">modifier le wikicode</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Les <a href="/wiki/Wikinews:Br%C3%A8ves" class="mw-redirect" title="Wikinews:Brèves">brèves Wikinews</a> consistent à formuler quelques courtes phrases sur un sujet d'actualité. Pour être synthétique, vous pouvez adopter la règle selon laquelle <span class="citation">«&#160;une phrase ne doit pas dépasser 20 mots&#160;»</span>.
</p>
<h2><span class="mw-headline" id="Les_articles">Les articles</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Aide:Comment_ajouter_du_contenu_avec_un_minimum_d%27effort&amp;veaction=edit&amp;section=2" class="mw-editsection-visualeditor" title="Modifier la section : Les articles">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Aide:Comment_ajouter_du_contenu_avec_un_minimum_d%27effort&amp;action=edit&amp;section=2" title="Modifier la section : Les articles">modifier le wikicode</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>L'article Wikinews présente des nouvelles synthétiques, généralement compris entre de 2000 à 5000 caractères. Vous pouvez utiliser les outils suivants pour le mettre en avant ou bien le protéger&#160;:
</p>
<ul><li>le placer dans l'<a href="/wiki/Wikinews:Article_principal" title="Wikinews:Article principal">article principal</a> de la page d'acceuil de Wikinews&#160;;</li>
<li>lorsque vous estimez que <a href="/w/index.php?title=Wikinews:Article_fini&amp;action=edit&amp;redlink=1" class="new" title="Wikinews:Article fini (page inexistante)">votre article Wikinews est fini</a> pour le mettre en semi-protection ({{fini}})&#160;;</li></ul>
<h2><span class="mw-headline" id="Les_dossiers">Les dossiers</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Aide:Comment_ajouter_du_contenu_avec_un_minimum_d%27effort&amp;veaction=edit&amp;section=3" class="mw-editsection-visualeditor" title="Modifier la section : Les dossiers">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Aide:Comment_ajouter_du_contenu_avec_un_minimum_d%27effort&amp;action=edit&amp;section=3" title="Modifier la section : Les dossiers">modifier le wikicode</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Les dossiers réunissent plusieurs articles sur une actualité encours (exemple&#160;: <a href="/wiki/Mod%C3%A8le:Dossier_type" title="Modèle:Dossier type">Modèle:Dossier type</a>)
</p>
<h2><span class="mw-headline" id="Les_portails">Les portails</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Aide:Comment_ajouter_du_contenu_avec_un_minimum_d%27effort&amp;veaction=edit&amp;section=4" class="mw-editsection-visualeditor" title="Modifier la section : Les portails">modifier</a><span class="mw-editsection-divider"> | </span><a href="/w/index.php?title=Aide:Comment_ajouter_du_contenu_avec_un_minimum_d%27effort&amp;action=edit&amp;section=4" title="Modifier la section : Les portails">modifier le wikicode</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Les portails, également appelés pages, traitent des actualités sur un secteur particulier (exemple&#160;: <a href="/wiki/Page:%C3%89v%C3%A8nements_scolaires" title="Page:Évènements scolaires">Page:Évènements scolaires</a>)
</p>

<h2><span class="mw-headline" id="Fonctionnement">Fonctionnement</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Wikinews:Supervision_de_qualit%C3%A9&amp;action=edit&amp;section=1" title="Modifier la section : Fonctionnement">modifier le wikicode</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Les visiteurs voient la dernière version stable (avec un lien vers le brouillon si des versions sont en attente de validation), tandis que les utilisateurs enregistrés voient directement le brouillon. Toute nouvelle modification doit être validée par un  relecteur pour être affichée dans la version stable.
</p>
<h2><span id="Utilisateurs_confirm.C3.A9s"></span><span class="mw-headline" id="Utilisateurs_confirmés">Utilisateurs confirmés</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Wikinews:Supervision_de_qualit%C3%A9&amp;action=edit&amp;section=2" title="Modifier la section : Utilisateurs confirmés">modifier le wikicode</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Les utilisateurs disposant du statut d'<a href="/wiki/Wikinews:Trusteduser" class="mw-redirect" title="Wikinews:Trusteduser">utilisateur confirmé</a> <small><span class="plainlinks"><a class="external text" href="https://fr.wikinews.org/w/index.php?title=Sp%C3%A9cial%3AListe_des_utilisateurs&amp;username=&amp;group=trusteduser&amp;limit=50">(liste)</a></span></small> voient leurs contributions automatiquement validées.
</p><p>Les utilisateurs disposant du statut de <a href="/wiki/Wikinews:Relecteurs" title="Wikinews:Relecteurs">relecteur</a> <small><span class="plainlinks"><a class="external text" href="https://fr.wikinews.org/w/index.php?title=Sp%C3%A9cial%3AListe_des_utilisateurs&amp;username=&amp;group=facilitator&amp;limit=50">(liste)</a></span></small> peuvent valider une modification pour qu'elle soit affichée dans la version stable.
</p><p>Ces deux statuts peuvent être attribués par un <a href="/wiki/Wikinews:Administrateur" class="mw-redirect" title="Wikinews:Administrateur">administrateur</a>.
</p>
<h2><span class="mw-headline" id="Relecture">Relecture</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Wikinews:Supervision_de_qualit%C3%A9&amp;action=edit&amp;section=3" title="Modifier la section : Relecture">modifier le wikicode</a><span class="mw-editsection-bracket">]</span></span></h2>
<h3><span id="Points_.C3.A0_v.C3.A9rifier_lors_de_la_validation_d.27une_modification"></span><span class="mw-headline" id="Points_à_vérifier_lors_de_la_validation_d'une_modification">Points à vérifier lors de la validation d'une modification</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Wikinews:Supervision_de_qualit%C3%A9&amp;action=edit&amp;section=4" title="Modifier la section : Points à vérifier lors de la validation d&#039;une modification">modifier le wikicode</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Il est suggéré au réviseur de vérifier les cinq points suivants avant de valider un nouvel article&#160;:
</p>
<ol><li><b>Droits d'auteurs <i>(Copyright)</i>&#160;:</b> le texte comme les images utilisées ne doivent pas porter atteinte au droit d'auteur.</li>
<li><b>Intérêt journalistique <i>(Contenu)</i>&#160;:</b> le texte publié doit constituer un <a href="https://fr.wikipedia.org/wiki/article_de_presse" class="extiw" title="w:article de presse">article de presse</a>. En adoptant <a href="/wiki/Wikinews:R%C3%A9daction_d%27un_article#Quels_sont_les_types_de_contributions_possibles_?" title="Wikinews:Rédaction d&#39;un article">l'une des différentes formes possibles sur Wikinews</a>. Les articles encyclopédiques, les éditoriaux ou les publicités sont proscrits.</li>
<li><b>Vérifiabilité <i>(Sources)</i>  :</b> à l'exception des <a href="/wiki/Wikinews:Reportage_original" title="Wikinews:Reportage original">reportages originaux</a> (qui doivent être clairement présentés comme tels), les informations présentes dans l'article doivent être confirmées par des sources (si possible indépendantes et multiples).</li>
<li><b><a href="/wiki/Wikinews:NPOV" class="mw-redirect" title="Wikinews:NPOV">Neutralité de point de vue</a> <i>(npov)</i> </b>&#160;: Les informations doivent être présentées de manière neutre, c'est-à-dire sans prendre parti pour un point de vue plutôt qu'un autre.</li>
<li><b>Conventions de style et de forme <i>(Style)</i>  :</b>  L'article doit respecter les conventions de <a href="/wiki/Wikinews:Conventions_de_style" title="Wikinews:Conventions de style">style</a>, de <a href="/wiki/Wikinews:Conventions_typographiques" title="Wikinews:Conventions typographiques">typographie</a>, de <a href="/wiki/Wikinews:Conventions_sur_les_titres" title="Wikinews:Conventions sur les titres">titrage</a> ainsi que les <a href="/wiki/Wikinews:Liens_inter-Wiki" class="mw-redirect" title="Wikinews:Liens inter-Wiki">liens inter-Wiki</a>, entre autre chose.</li></ol>
<ul><li><b>Si les critères sont respectés</b>, le relecteur doit retirer le modèle <a href="/wiki/Mod%C3%A8le:Relecture" title="Modèle:Relecture">{{Relecture}}</a> et ajouter le modèle <a href="/wiki/Mod%C3%A8le:Publication" title="Modèle:Publication">{{Publication}}</a> au bas de la page, si ce n'est déjà fait.</li></ul>
<ul><li><b>Si l'un de ces critères n'est pas respecté</b>, le réviseur peut simplement corriger l'article ou bien reporter ses remarques en page de discussion en utilisant, s'il le souhaite, le modèle <a href="/wiki/Mod%C3%A8le:R%C3%A9vision_par_des_pairs" title="Modèle:Révision par des pairs">{{Révision par des pairs}}</a> (ou <a href="/wiki/Mod%C3%A8le:RPP" class="mw-redirect" title="Modèle:RPP">{{RPP}}</a>) qui permet de montrer clairement les points à améliorer d'un article. Consulter la <a href="/wiki/Aide:Relire_un_article" title="Aide:Relire un article">page d'aide</a> pour plus de détails.</li></ul>
<p>Veuillez noter que seuls les éditeurs enregistrés avec le statut de <a href="/wiki/Wikinews:Relecteurs" title="Wikinews:Relecteurs">relecteur</a> peuvent relire et publier des articles. Pour acquérir ce droit, vous pouvez faire une requête <a href="/wiki/Wikinews:Requ%C3%AAte_aux_administrateurs" title="Wikinews:Requête aux administrateurs">sur cette page</a>.
</p>
<h3><span class="mw-headline" id="Publication">Publication</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Wikinews:Supervision_de_qualit%C3%A9&amp;action=edit&amp;section=5" title="Modifier la section : Publication">modifier le wikicode</a><span class="mw-editsection-bracket">]</span></span></h3>
<p>Pour publier une page, remplacez le modèle <code>{{Développement}}</code> ou <code>{{Relecture}}</code> par <code>{{Publication|jour={{subst:CURRENTDAY}}|mois={{subst:CURRENTMONTHNAME}}|année={{subst:CURRENTYEAR}}}}</code>. Soit manuellement, soit en cliquant sur le modèle sous la boîte d'édition (sous les caractères spéciaux). S'il s'agit d'une <a href="/wiki/Wikinews:Br%C3%A8ves" class="mw-redirect" title="Wikinews:Brèves">brève</a>, ajouter le paramètre <code>|statut=brève</code>
</p>
